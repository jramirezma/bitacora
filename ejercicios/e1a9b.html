El problema

Sea $f$ una función tal que, para $x \in \mathbb{R}$, $$3f(2-x) + 2f(x) = x^2.$$ Calcule $f(0)$, $f(1)$ y $f(2)$.

¿De qué se trata esto?

Tenemos una ecuación que relaciona $f(x)$ con $f(2-x)$. Eso significa que $f$ evaluada en $x$ y $f$ evaluada en $2-x$ están atadas por esa ecuación. Solos, ninguno de los dos valores se puede despejar. Pero si usamos la ecuación dos veces, una con $x$ y otra con $2-x$ en lugar de $x$, obtenemos dos ecuaciones con dos incógnitas: un sistema.

El truco es sustituir $x$ por $2-x$ en la ecuación original. Eso genera una segunda ecuación, y con las dos juntas se puede despejar $f(x)$.

Una vez que tenemos $f(x)$ en forma general, calcular $f(0)$, $f(1)$ y $f(2)$ es simplemente sustituir.

La función no quería revelar su fórmula con una sola ecuación. Hay que preguntarle dos veces.

Manos a la obra

Paso 1: plantear el sistema

Desarrollo
Justificación
$3f(2-x) + 2f(x) = x^2 \quad \text{(I)}$
Ecuación original.
$3f(2-(2-x)) + 2f(2-x) = (2-x)^2$
Se sustituye $x \leftarrow 2-x$ en la ecuación original. Vale para todo $x \in \mathbb{R}$.
$3f(x) + 2f(2-x) = (2-x)^2 \quad \text{(II)}$
Se simplifica $2-(2-x) = x$.

Paso 2: resolver el sistema para f(x)

Desarrollo
Justificación
$2 \times \text{(I)}: \quad 6f(2-x) + 4f(x) = 2x^2$
Se multiplica (I) por 2 para igualar el coeficiente de $f(2-x)$ con la siguiente ecuación.
$3 \times \text{(II)}: \quad 6f(2-x) + 9f(x) = 3(2-x)^2$
Se multiplica (II) por 3.
$3\times\text{(II)} - 2\times\text{(I)}:$ $5f(x) = 3(2-x)^2 - 2x^2$
Se restan para eliminar $f(2-x)$. Queda solo $f(x)$.
$5f(x) = 3(4 - 4x + x^2) - 2x^2$
Se expande $(2-x)^2 = 4 - 4x + x^2$.
$5f(x) = 12 - 12x + 3x^2 - 2x^2$
Se distribuye el 3.
$5f(x) = x^2 - 12x + 12$
Se agrupan los términos semejantes.
$f(x) = \dfrac{x^2 - 12x + 12}{5}$
Se divide por 5.

Paso 3: calcular los valores pedidos

Desarrollo
Justificación
$f(0) = \dfrac{0 - 0 + 12}{5} = \dfrac{12}{5}$
Se sustituye $x = 0$.
$f(1) = \dfrac{1 - 12 + 12}{5} = \dfrac{1}{5}$
Se sustituye $x = 1$.
$f(2) = \dfrac{4 - 24 + 12}{5} = \dfrac{-8}{5}$
Se sustituye $x = 2$.

Chequeemos que esto tiene sentido

Verificamos con $x = 0$ en la ecuación original $3f(2-x) + 2f(x) = x^2$:

$$3f(2) + 2f(0) = 3\cdot\frac{-8}{5} + 2\cdot\frac{12}{5} = \frac{-24}{5} + \frac{24}{5} = 0 = 0^2 \checkmark$$

Verificamos con $x = 1$:

$$3f(1) + 2f(1) = 5\cdot\frac{1}{5} = 1 = 1^2 \checkmark$$
Resultado. La función es $\displaystyle f(x) = \frac{x^2 - 12x + 12}{5}$, y los valores pedidos son: $$\boxed{f(0) = \frac{12}{5}, \qquad f(1) = \frac{1}{5}, \qquad f(2) = -\frac{8}{5}}$$

¿Dónde ya existe esto?

Lo que hicimos acá es resolver un sistema de dos ecuaciones con dos incógnitas, solo que las incógnitas eran valores de una función en lugar de números sueltos. Esa misma idea aparece en el corazón de los algoritmos de recomendación: cuando Netflix o Spotify quieren estimar cuánto le va a gustar una canción a un usuario, no tienen el valor directo, pero tienen muchas ecuaciones que relacionan ese valor desconocido con otros valores conocidos. Resolver el sistema, igual que acá, es la forma de extraer la información que no estaba dada explícitamente.
