Ejercicio

Halle una cota superior para:

a) $A = \{x - 2x^2 : x \in \mathbb{R}\}$.

b) $B = \{|x+3| - |x| + 2 : x \in \mathbb{R}\}$.

c) $C = \{x : \sqrt{x+2} < 2\}$.

Análisis

Una cota superior de un conjunto $S$ es cualquier número $M$ tal que $s \leq M$ para todo $s \in S$. No tiene que ser el mínimo de tales cotas (eso sería el supremo); basta con encontrar alguna. Para a) completamos el cuadrado. Para b) usamos la desigualdad triangular. Para c) resolvemos la inecuación y leemos el extremo superior.

No se pide el supremo exacto, sino cualquier cota superior válida. Dicho eso, lo más natural es encontrar la cota más ajustada posible, que coincide con el supremo.

Solución — a) $\ A = \{x-2x^2 : x\in\mathbb{R}\}$

Desarrollo
Justificación
$$x - 2x^2 = -2\!\left(x^2 - \frac{x}{2}\right)$$
Factor $-2$
Factorizamos para preparar el cuadrado completo.
$$= -2\!\left(x^2 - \frac{x}{2} + \frac{1}{16} - \frac{1}{16}\right)$$ $$= -2\!\left(x - \frac{1}{4}\right)^2 + \frac{1}{8}$$
Completar el cuadrado
Sumamos y restamos $\left(\frac{1}{4}\right)^2 = \frac{1}{16}$. $-2 \cdot (-\frac{1}{16}) = \frac{1}{8}$.
$$-2\!\left(x-\tfrac{1}{4}\right)^2 + \frac{1}{8} \leq \frac{1}{8}$$
Cuadrado no negativo
$-2(x-\frac{1}{4})^2 \leq 0$ para todo $x$. El máximo se alcanza en $x = \frac{1}{4}$.
$$\boxed{M = \frac{1}{8} \text{ es cota superior de } A}$$
Conclusión
Es además el supremo y máximo de $A$, alcanzado en $x = \frac{1}{4}$.

Solución — b) $\ B = \{|x+3|-|x|+2 : x\in\mathbb{R}\}$

Desarrollo
Justificación
$$|x+3| - |x| \leq |x+3-x| = |3| = 3$$
Desigualdad triangular inversa
$|A| - |B| \leq |A - B|$. Con $A = x+3$ y $B = x$: $|x+3| - |x| \leq |(x+3)-x| = 3$.
$$|x+3| - |x| + 2 \leq 3 + 2 = 5$$
Suma de 2
Sumamos $2$ a ambos lados. La igualdad se alcanza cuando $x \geq 0$ (ahí $|x+3|-|x| = x+3-x = 3$). Por ejemplo $x=0$: $|3|-|0|+2 = 5$ ✓.
$$\boxed{M = 5 \text{ es cota superior de } B}$$
Conclusión
Es además el supremo y máximo de $B$.

Solución — c) $\ C = \{x : \sqrt{x+2} < 2\}$

Desarrollo
Justificación
$$\text{Dominio: } x+2 \geq 0 \;\Rightarrow\; x \geq -2$$
Dominio de la raíz
El radicando debe ser no negativo.
$$\sqrt{x+2} < 2 \;\Leftrightarrow\; x+2 < 4 \;\Rightarrow\; x < 2$$
Elevar al cuadrado
Ambos lados no negativos. $(\sqrt{x+2})^2 = x+2$ y $2^2 = 4$.
$$C = [-2,\, 2)$$
Descripción del conjunto
Combinamos dominio ($x \geq -2$) con la inecuación ($x < 2$).
$$\boxed{M = 2 \text{ es cota superior de } C}$$
Conclusión
Todo $x \in C$ satisface $x < 2$, así que $2$ es cota superior. Es el supremo de $C$ pero no el máximo (no pertenece a $C$).
Resultado.
a) $M = \dfrac{1}{8}$ es cota superior (y máximo) de $A$. Se alcanza en $x = \dfrac{1}{4}$.

b) $M = 5$ es cota superior (y máximo) de $B$. Se alcanza para todo $x \geq 0$.

c) $M = 2$ es cota superior (y supremo) de $C = [-2,2)$. No es máximo pues $2 \notin C$.
