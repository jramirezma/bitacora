Ejercicio

a) Identificar 5 clases donde cada clase debe tener 5 atributos significativos.

b) Implementar dos clases definidas en el inciso a.

b.1) Instanciar dos objetos para cada clase.

b.2) Para los objetos definidos, comparar dos atributos de cada clase a través de un método.

Contexto: Carnavales de Bolivia.

Análisis

El ejercicio tiene dos partes. Primero hay que modelar el dominio: pensar qué entidades participan en los carnavales y qué datos las describen. Luego hay que implementar dos de esas entidades en pseudocódigo, creando objetos y comparando sus atributos mediante un método propio de la clase.

La comparación de atributos debe hacerse desde dentro de la clase, usando un método compararCon() que recibe otro objeto del mismo tipo y accede a sus datos mediante getters.

Para el inciso a se identifican cinco clases del mundo del carnaval boliviano: Bailarín, Comparsa, Banda Musical, Espectador y Disfraz. Cada una tiene cinco atributos que la describen de forma significativa. Para el inciso b se implementan Disfraz y Bailarín, comparando respectivamente precio/talla y edad/años de experiencia.

Cinco clases, cinco atributos, dos implementaciones: parece el número de comparsas que entran por la misma calle al mismo tiempo en Oruro.

Clase: Disfraz

Justificación
class Disfraz
{
    String nombre, color, material, talla;
    float  precio;
Declaración de atributos
Se declaran los cinco atributos significativos de la clase. precio es float porque puede tener decimales; los demás son cadenas de texto. 
    public:

    leer()
    {
        read(nombre, color, material,
             talla, precio);
    }

    mostrar()
    {
        print(nombre, color, material,
              talla, precio);
    }
Métodos de entrada/salida
leer() carga los datos desde el usuario. mostrar() los imprime. Son la interfaz básica del objeto con el exterior. 
    String getNombre() { return nombre; }
    float  getPrecio() { return precio; }
    String getTalla()  { return talla; }
Getters
Permiten que otros objetos accedan a los atributos sin romper el encapsulamiento. El método compararCon() los usará para leer los datos del objeto recibido como parámetro. 
    // b.2) Compara precio y talla
    compararCon(Disfraz otro)
    {
        if (precio > otro.getPrecio()) then
            print(nombre + " es mas caro");
        elseif (precio < otro.getPrecio()) then
            print(otro.getNombre()
                  + " es mas caro");
        else
            print("Mismo precio");

        if (talla == otro.getTalla()) then
            print("Misma talla");
        else
            print("Talla distinta");
    }
}
Método compararCon()
Compara los dos atributos solicitados: precio (numérico, tres casos: mayor, menor, igual) y talla (cadena, dos casos: igual o distinta). El objeto otro se consulta mediante sus getters. 
Begin
    d1 = creaobjeto Disfraz();
    d2 = creaobjeto Disfraz();
    d1.leer();
    d2.leer();
    d1.mostrar();
    d2.mostrar();
    d1.compararCon(d2);
End
Programa principal
Se instancian dos objetos (b.1), se cargan y muestran sus datos, y luego d1 invoca la comparación contra d2 (b.2).

Getters
Los tres getters exponen los atributos que necesita el método compararCon(): nombre (para los mensajes) y los dos atributos numéricos a comparar. 
    // b.2) Compara edad y aniosExperiencia

Método compararCon()
Compara edad y años de experiencia, ambos enteros con tres casos posibles cada uno. La estructura es idéntica a la de Disfraz: mayor, menor o igual.
