Ejercicio

Encuentre (si existe) el supremo, ínfimo, máximo y mínimo de los conjuntos:

a) $A = \left\{\dfrac{1}{n}+1 : n\in\mathbb{N}\right\}$.

b) $B = \left\{\dfrac{2n+1}{n+1} : n\in\mathbb{N}\right\}$.

c) $C = \left\{\dfrac{3n^2-n+3}{n^2+1} : n\in\mathbb{N}\right\}$.

d) $D = \left\{1+\dfrac{2}{3}+\dfrac{2^2}{3^2}+\cdots+\dfrac{2^{n-1}}{3^{n-1}} : n\in\mathbb{N}\right\}$.

Análisis

Para cada conjunto hay que estudiar el comportamiento de la sucesión que lo define: si es creciente o decreciente, y hacia dónde tiende cuando $n \to \infty$. El mínimo o máximo existe si la sucesión alcanza su extremo en algún $n$ concreto. El supremo o ínfimo existe si el conjunto está acotado, aunque no se alcance.

Si $f(n)$ es decreciente: $\min = \lim_{n\to\infty} f(n)$ (si se alcanza) o ínfimo, y $\max = f(1)$. Si es creciente: al revés. Verificar si el límite pertenece al conjunto.

Usamos $\mathbb{N} = \{1, 2, 3, \ldots\}$.

Solución — a) $\ A = \left\{\dfrac{1}{n}+1\right\}$

Desarrollo
Justificación
$$a_n = \frac{1}{n}+1$$ $$a_1 = 2,\quad a_2 = \tfrac{3}{2},\quad a_3 = \tfrac{4}{3},\quad \ldots$$
Primeros valores
La sucesión empieza en $2$ y decrece hacia $1$.
$$a_{n+1} - a_n = \frac{1}{n+1} - \frac{1}{n} = \frac{-1}{n(n+1)} < 0$$
Monotonía
$a_{n+1} < a_n$ para todo $n$: la sucesión es estrictamente decreciente.
$$\lim_{n\to\infty} \left(\frac{1}{n}+1\right) = 1$$ $$1 \notin A \quad(\text{nunca se alcanza})$$
Límite
$\frac{1}{n} \to 0$, pero $\frac{1}{n} > 0$ para todo $n \in \mathbb{N}$, así que $a_n > 1$ siempre: el valor $1$ no pertenece a $A$.
$$\sup A = \max A = 2 = a_1$$ $$\inf A = 1, \quad \min A \text{ no existe}$$
Conclusión
El máximo $2$ se alcanza en $n=1$. El ínfimo $1$ no se alcanza.

Solución — b) $\ B = \left\{\dfrac{2n+1}{n+1}\right\}$

Desarrollo
Justificación
$$b_n = \frac{2n+1}{n+1} = 2 - \frac{1}{n+1}$$
Reescritura
División: $\frac{2n+1}{n+1} = \frac{2(n+1)-1}{n+1} = 2 - \frac{1}{n+1}$. Esto muestra claramente la monotonía.
$$b_1 = \tfrac{3}{2},\quad b_2 = \tfrac{5}{3},\quad b_3 = \tfrac{7}{4},\quad \ldots$$ $$b_{n+1} - b_n = \frac{1}{n+1} - \frac{1}{n+2} > 0$$
Monotonía
$b_{n+1} > b_n$: la sucesión es estrictamente creciente.
$$\lim_{n\to\infty} \left(2-\frac{1}{n+1}\right) = 2$$ $$2 \notin B \quad(\text{nunca se alcanza})$$
Límite
$b_n < 2$ para todo $n$ (pues $\frac{1}{n+1} > 0$). El valor $2$ no pertenece a $B$.
$$\inf B = \min B = \frac{3}{2} = b_1$$ $$\sup B = 2, \quad \max B \text{ no existe}$$
Conclusión
El mínimo $\frac{3}{2}$ se alcanza en $n=1$. El supremo $2$ no se alcanza.

Solución — c) $\ C = \left\{\dfrac{3n^2-n+3}{n^2+1}\right\}$

Desarrollo
Justificación
$$c_n = \frac{3n^2-n+3}{n^2+1} = 3 - \frac{n}{n^2+1}$$
Reescritura
$\frac{3n^2-n+3}{n^2+1} = \frac{3(n^2+1)-n}{n^2+1} = 3 - \frac{n}{n^2+1}$.
$$c_1 = \tfrac{5}{2},\quad c_2 = \tfrac{13}{5},\quad c_3 = \tfrac{27}{10},\quad c_4 = \tfrac{47}{17}$$
Primeros valores
$c_1 = 2.5$, $c_2 = 2.6$, $c_3 = 2.7$, $c_4 \approx 2.76$. Parece creciente al inicio.
$$g(n) = \frac{n}{n^2+1}: \quad g'(x) = \frac{1-x^2}{(x^2+1)^2}$$ $$g'(x) < 0 \text{ para } x > 1$$
Monotonía de $g(n) = \frac{n}{n^2+1}$
Tratamos $n$ como variable continua. $g$ crece en $(0,1)$ y decrece en $(1,\infty)$. Para $n \geq 2$ entero, $g$ es decreciente, así que $c_n = 3-g(n)$ es creciente para $n \geq 2$.
$$g(1) = \frac{1}{2},\quad g(2) = \frac{2}{5},\quad g(3)=\frac{3}{10},\ldots$$ $$g(1) > g(2) > g(3) > \cdots \to 0$$ $$\Rightarrow c_1 < c_2 < c_3 < \cdots \to 3$$
Comportamiento global
$g(n)$ decrece para $n \geq 1$ (verificando: $g(1)=0.5 > g(2)=0.4$). Entonces $c_n$ es creciente para todo $n \geq 1$. El mínimo es $c_1$ y el supremo es $3$.
$$\lim_{n\to\infty} c_n = 3, \quad 3 \notin C$$ $$\inf C = \min C = c_1 = \frac{5}{2}$$ $$\sup C = 3, \quad \max C \text{ no existe}$$
Conclusión
$c_n < 3$ para todo $n$ (pues $g(n) > 0$). El mínimo $\frac{5}{2}$ se alcanza en $n=1$.

Solución — d) $\ D = \left\{1+\dfrac{2}{3}+\cdots+\dfrac{2^{n-1}}{3^{n-1}}\right\}$

Desarrollo
Justificación
$$d_n = \sum_{k=0}^{n-1}\left(\frac{2}{3}\right)^k = \frac{1-\left(\frac{2}{3}\right)^n}{1-\frac{2}{3}} = 3\left(1-\left(\frac{2}{3}\right)^n\right)$$
Suma de progresión geométrica
Razón $r = \frac{2}{3}$, primer término $1$, $n$ términos. Fórmula: $S_n = \frac{1-r^n}{1-r}$.
$$d_1 = 1,\quad d_2 = \tfrac{5}{3},\quad d_3 = \tfrac{19}{9},\quad \ldots$$ $$d_{n+1} - d_n = \left(\frac{2}{3}\right)^n > 0$$
Monotonía
Cada término agrega $\left(\frac{2}{3}\right)^n > 0$: la sucesión es estrictamente creciente.
$$\lim_{n\to\infty} d_n = 3\left(1 - 0\right) = 3$$ $$d_n < 3 \text{ para todo } n \quad(\text{pues } \left(\tfrac{2}{3}\right)^n > 0)$$ $$3 \notin D$$
Límite
$\left(\frac{2}{3}\right)^n \to 0$ pero nunca vale $0$, así que $d_n < 3$ siempre.
$$\inf D = \min D = d_1 = 1$$ $$\sup D = 3, \quad \max D \text{ no existe}$$
Conclusión
El mínimo $1$ se alcanza en $n=1$. El supremo $3$ no se alcanza.
Resultado.

a) $\sup A = \max A = 2$; $\quad\inf A = 1$, $\min A$ no existe.

b) $\inf B = \min B = \dfrac{3}{2}$; $\quad\sup B = 2$, $\max B$ no existe.

c) $\inf C = \min C = \dfrac{5}{2}$; $\quad\sup C = 3$, $\max C$ no existe.

d) $\inf D = \min D = 1$; $\quad\sup D = 3$, $\max D$ no existe.
