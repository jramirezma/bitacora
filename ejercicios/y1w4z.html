Ejercicio

Resuelva:

a) $\sqrt{x-2} < 3$.    b) $\sqrt{10x-x^2} < 3$.    c) $\sqrt{x^2-10x} \geq 3$.

Análisis

Al trabajar con raíces hay dos pasos obligatorios: primero establecer el dominio (el radicando debe ser $\geq 0$), y luego resolver la inecuación. Para elevar al cuadrado y preservar el sentido, ambos lados deben ser no negativos.

$\sqrt{E} < c$ con $c > 0$ equivale a $0 \leq E < c^2$. $\sqrt{E} \geq c$ con $c \geq 0$ equivale a $E \geq c^2$ (y $E \geq 0$, que ya está garantizado).

Solución — a) $\ \sqrt{x-2} < 3$

Desarrollo
Justificación
$$\text{Dominio: } x - 2 \geq 0 \;\Rightarrow\; x \geq 2$$
Dominio
El radicando debe ser no negativo para que la raíz esté definida.
$$\sqrt{x-2} < 3 \;\Leftrightarrow\; x-2 < 9$$ $$x < 11$$
Elevar al cuadrado
Como $\sqrt{x-2} \geq 0$ y $3 > 0$, elevar al cuadrado preserva el sentido. $\sqrt{x-2} < 3 \Leftrightarrow x-2 < 9$.
$$x \geq 2 \cap x < 11$$ $$\boxed{x \in [2,\, 11)}$$
Intersección con dominio
Combinamos la condición de dominio con la solución de la inecuación.

Solución — b) $\ \sqrt{10x-x^2} < 3$

Desarrollo
Justificación
$$10x - x^2 \geq 0 \;\Rightarrow\; x(10-x) \geq 0$$ $$0 \leq x \leq 10$$
Dominio
$x(10-x) \geq 0$ cuando ambos factores tienen el mismo signo. Esto ocurre para $x \in [0,10]$.
$$\sqrt{10x-x^2} < 3 \;\Leftrightarrow\; 10x-x^2 < 9$$ $$x^2-10x+9 > 0 \;\Rightarrow\; (x-1)(x-9) > 0$$ $$x < 1 \text{ o } x > 9$$
Elevar al cuadrado
Ambos lados no negativos en el dominio. Reorganizamos y factorizamos. El producto $(x-1)(x-9) > 0$ fuera de las raíces $1$ y $9$.
$$[0,10] \cap (x<1 \text{ o } x>9)$$ $$= [0,1)\cup(9,10]$$ $$\boxed{x \in [0,1)\cup(9,10]}$$
Intersección con dominio
Cruzamos con $[0,10]$. En $x=0$ y $x=10$ la raíz vale $0 < 3$, se incluyen. En $x=1$ y $x=9$ la raíz vale $3$, no satisface la estricta, se excluyen.

Solución — c) $\ \sqrt{x^2-10x} \geq 3$

Desarrollo
Justificación
$$x^2-10x \geq 0 \;\Rightarrow\; x(x-10) \geq 0$$ $$x \leq 0 \text{ o } x \geq 10$$
Dominio
$x(x-10) \geq 0$ fuera del intervalo $(0,10)$.
$$\sqrt{x^2-10x} \geq 3 \;\Leftrightarrow\; x^2-10x \geq 9$$ $$x^2-10x-9 \geq 0$$ $$x = \frac{10\pm\sqrt{100+36}}{2} = 5\pm\sqrt{34}$$
Elevar al cuadrado
$\sqrt{E} \geq 3 \Leftrightarrow E \geq 9$. Resolvemos $x^2-10x-9 \geq 0$. Raíces: $x = 5\pm\sqrt{34}$ ($\sqrt{34}\approx 5.83$).
$$x \leq 5-\sqrt{34} \text{ o } x \geq 5+\sqrt{34}$$

$5-\sqrt{34} \approx -0.83$ y $5+\sqrt{34}\approx 10.83$.

Solución de la inecuación cuadrática
El cuadrado abre hacia arriba, es $\geq 0$ fuera de las raíces.
$$(x\leq 0\text{ o }x\geq 10)\cap(x\leq 5-\sqrt{34}\text{ o }x\geq 5+\sqrt{34})$$ $$= (-\infty,\,5-\sqrt{34}\,]\cup[\,5+\sqrt{34},\,+\infty)$$
Intersección con dominio
Como $5-\sqrt{34}<0$ y $5+\sqrt{34}>10$, las condiciones del dominio quedan automáticamente cubiertas por la inecuación cuadrática.
$$\boxed{x \in \left(-\infty,\,5-\sqrt{34}\,\right]\cup\left[\,5+\sqrt{34},\,+\infty\right)}$$
Solución
Extremos incluidos por $\geq$.
Resultado.
a) $x \in [2,\,11)$.
b) $x \in [0,1)\cup(9,10]$.
c) $x \in \left(-\infty,\,5-\sqrt{34}\right]\cup\left[5+\sqrt{34},\,+\infty\right)$.

En todos los casos: primero dominio, luego elevar al cuadrado, finalmente intersectar.
