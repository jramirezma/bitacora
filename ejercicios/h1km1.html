El problema

Resolver la inecuación: $$\frac{\sqrt{x^2 - 4x + 4}}{-x} < 2$$

¿Qué hay que ver acá?

Antes de operar, hay dos cosas que resolver. Primero: el radicando $x^2 - 4x + 4$ es un trinomio cuadrado perfecto, igual a $(x-2)^2$. Eso significa que la raíz es $\sqrt{(x-2)^2} = |x-2|$. Segundo: el denominador es $-x$, que puede ser positivo o negativo según el signo de $x$. Como el signo del denominador cambia la dirección de la desigualdad al multiplicar, hay que analizar los dos casos por separado.

Cuando se multiplica o divide una inecuación por una expresión cuyo signo no es fijo, hay que separar en casos. En cada caso, si el factor es positivo la desigualdad se mantiene; si es negativo, se invierte.

El dominio exige $-x \neq 0$, es decir $x \neq 0$. Y la raíz siempre existe porque $(x-2)^2 \geq 0$ para todo $x$. Entonces el dominio es $\mathbb{R} \setminus \{0\}$, y ahí se divide en $x < 0$ y $x > 0$.

Una raíz que esconde un cuadrado perfecto adentro. El examen avisa que hay que mirar antes de calcular.

Dato de color

La identidad $\sqrt{x^2} = |x|$ parece obvia pero tiene una trampa clásica: mucha gente escribe $\sqrt{x^2} = x$ sin pensar. Eso es verdad solo cuando $x \geq 0$. Para $x < 0$, $\sqrt{x^2}$ da un número positivo, pero $x$ es negativo: no pueden ser iguales. El valor absoluto existe exactamente para capturar ese comportamiento: $|x|$ siempre devuelve la distancia al cero, sin importar el signo de $x$.

La demostración

Simplificación previa

Desarrollo
Justificación
$x^2 - 4x + 4 = (x-2)^2$
Trinomio cuadrado perfecto: $(a-b)^2 = a^2 - 2ab + b^2$ con $a=x$, $b=2$.
$\sqrt{(x-2)^2} = |x-2|$
$\sqrt{u^2} = |u|$ para todo $u \in \mathbb{R}$.
La inecuación queda: $\dfrac{|x-2|}{-x} < 2$
Se sustituye la raíz simplificada. Dominio: $x \neq 0$.

Caso 1 — $x < 0$

Desarrollo
Justificación
$x < 0 \Rightarrow -x > 0$
El denominador es positivo en este caso.
$|x-2| < 2 \cdot (-x)$
Se multiplica ambos lados por $-x > 0$: la desigualdad se mantiene.
$|x-2| < -2x$
Se simplifica el lado derecho.
$-(-2x) < x - 2 < -2x$
Definición de valor absoluto: $|u| < k \Leftrightarrow -k < u < k$, con $k = -2x > 0$.
$2x < x - 2 < -2x$
Se simplifica $-(-2x) = 2x$.
De $2x < x - 2$: $\quad x < -2$
Se resta $x$ en ambos lados.
De $x - 2 < -2x$: $\quad 3x < 2 \Rightarrow x < \tfrac{2}{3}$
Se suma $2x$ y se suma $2$ en ambos lados.
Intersección con $x < 0$: $\quad x < -2$
Las tres condiciones juntas: $x < -2$ es la más restrictiva dentro de $x < 0$.

Caso 2 — $x > 0$

Desarrollo
Justificación
$x > 0 \Rightarrow -x < 0$
El denominador es negativo en este caso.
$|x-2| > 2 \cdot (-x)$
Se multiplica ambos lados por $-x < 0$: la desigualdad se invierte.
$|x-2| > -2x$
Se simplifica el lado derecho.
Para $x > 0$: $-2x < 0$
El lado derecho es negativo.
$|x-2| \geq 0 > -2x$ para todo $x > 0$
El valor absoluto es siempre $\geq 0$, que ya supera a cualquier número negativo.
La desigualdad se cumple para todo $x > 0$
No hay restricción adicional dentro de este caso.

Comprobemos con números

Caso 1 — un punto que debería cumplir: $x = -3$:

$\dfrac{\sqrt{9+12+4}}{3} = \dfrac{\sqrt{25}}{3} = \dfrac{5}{3} \approx 1.67 < 2$ ✓

Caso 1 — un punto que no debería cumplir: $x = -1$ (está entre $-2$ y $0$):

$\dfrac{\sqrt{1+4+4}}{1} = \dfrac{3}{1} = 3 \not< 2$ ✓

Caso 2 — un punto que debería cumplir: $x = 1$:

$\dfrac{\sqrt{1-4+4}}{-1} = \dfrac{1}{-1} = -1 < 2$ ✓

Resultado. El conjunto solución es: $$\boxed{x \in (-\infty,\, -2) \cup (0,\, +\infty)}$$ Es decir: todos los $x$ menores que $-2$, y todos los $x$ positivos.

¿Ya usé esto sin saberlo?

La separación por casos según el signo del denominador es exactamente lo que hace cualquier función que necesita saber si dividir "da vuelta" el resultado. En programación, cuando se compara el resultado de una división y el divisor puede ser negativo, ignorar ese signo produce bugs difíciles de detectar: la comparación parece correcta en los tests con valores positivos y falla silenciosamente con negativos. Las funciones de ordenamiento que usan un comparador personalizado tienen este problema con frecuencia: si el comparador devuelve un valor cuyo signo depende de la entrada, la función de sort puede producir resultados incorrectos sin lanzar ningún error.

Preguntas para explorar

  1. ¿Por qué $\sqrt{x^2} = |x|$ y no simplemente $x$? ¿Para qué valores de $x$ coinciden $\sqrt{x^2}$ y $x$?
  2. Al resolver $\frac{|x-2|}{-x} < 2$, en el caso $x > 0$ la desigualdad se cumple para todo $x$ positivo. ¿Eso siempre pasa cuando el lado derecho tras multiplicar es negativo, o depende de la expresión concreta?
  3. ¿Qué cambia en la solución si la inecuación fuera $\frac{\sqrt{x^2-4x+4}}{-x} \leq 2$ (con $\leq$ en lugar de $<$)?
  4. ¿Cómo se resolvería la misma inecuación $\frac{\sqrt{x^2-4x+4}}{-x} < 2$ si el radicando fuera $x^2 - 4x + 3$ en lugar de $x^2 - 4x + 4$? ¿Qué cambia en el procedimiento?
  5. Al multiplicar ambos lados de una inecuación por una expresión que puede cambiar de signo, ¿qué error se comete si no se separa en casos? ¿Se puede dar un ejemplo concreto donde ese error lleve a una solución incorrecta?
