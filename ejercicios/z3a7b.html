El problema

Determine si la función es creciente o decreciente en su dominio: $$h(x) = x^3$$

¿De qué se trata esto?

El esquema es el mismo de siempre: tomamos $x_1 < x_2$ y analizamos el signo de $h(x_1) - h(x_2) = x_1^3 - x_2^3$. La diferencia de cubos tiene una factorización conocida que nos va a permitir analizar el signo directamente.

Diferencia de cubos: $a^3 - b^3 = (a - b)(a^2 + ab + b^2)$. Una vez factorizada, se analiza el signo de cada factor por separado.

El factor $a^2 + ab + b^2$ merece atención: no es obvio que siempre sea positivo. Hay que demostrarlo.

$x^3$ crece sin freno en ambas direcciones. Pero hay que probarlo, no solo dibujarlo en la cabeza.

Manos a la obra

Desarrollo
Justificación
Sean $x_1, x_2 \in \mathbb{R}$ con $x_1 < x_2$
Se toman dos valores arbitrarios con $x_1 < x_2$. El dominio es todo $\mathbb{R}$.
$h(x_1) - h(x_2) = x_1^3 - x_2^3$
Se sustituye la definición de $h$.
$= (x_1 - x_2)(x_1^2 + x_1 x_2 + x_2^2)$
Factorización de diferencia de cubos: $a^3 - b^3 = (a-b)(a^2+ab+b^2)$.
$x_1 < x_2 \Rightarrow x_1 - x_2 < 0$
El primer factor es negativo por hipótesis.
$x_1^2 + x_1 x_2 + x_2^2$ $= \left(x_1 + \dfrac{x_2}{2}\right)^2 + \dfrac{3x_2^2}{4}$
Se completa el cuadrado para analizar el signo del segundo factor.
$\left(x_1 + \dfrac{x_2}{2}\right)^2 \geq 0$ y $\dfrac{3x_2^2}{4} \geq 0$
Todo cuadrado es mayor o igual a cero.
$x_1^2 + x_1 x_2 + x_2^2 \geq 0$
Suma de dos términos no negativos.
$x_1^2 + x_1 x_2 + x_2^2 = 0 \iff x_1 = x_2 = 0$
El único caso donde ambos sumandos son cero a la vez es $x_1 = x_2 = 0$, pero eso contradice $x_1 < x_2$. Entonces el segundo factor es estrictamente positivo.
$x_1^2 + x_1 x_2 + x_2^2 > 0$
Confirmado: el segundo factor es siempre positivo cuando $x_1 \neq x_2$.
$(x_1 - x_2)(x_1^2 + x_1 x_2 + x_2^2) < 0$
Negativo por positivo da negativo.
$h(x_1) < h(x_2)$
Cuando $x_1 < x_2$, el valor de $h$ sube. Por definición, $h$ es creciente.

¿Tiene sentido lo que obtuvimos?

Tomamos un par de valores en distintas regiones para cubrir el caso donde los dos son negativos, que es el menos intuitivo:

$x_1 = -3 < x_2 = -1$: $\quad h(-3) = -27 < -1 = h(-1)$ ✓

$x_1 = -2 < x_2 = 2$: $\quad h(-2) = -8 < 8 = h(2)$ ✓

$x_1 = 1 < x_2 = 4$: $\quad h(1) = 1 < 64 = h(4)$ ✓

Resultado. $h(x) = x^3$ es $\boxed{\text{estrictamente creciente}}$ en todo $\mathbb{R}$.

Para todo $x_1, x_2 \in \mathbb{R}$ con $x_1 < x_2$ se tiene $h(x_1) < h(x_2)$.

¿Dónde ya existe esto?

Que $x^3$ sea estrictamente creciente en todo $\mathbb{R}$ es lo que garantiza que la función cúbica sea invertible: como nunca repite valores, siempre se puede recuperar el $x$ original a partir de $x^3$. Esa propiedad se usa en criptografía de curva elíptica, donde las operaciones se definen sobre puntos de una curva cúbica y la invertibilidad es la base de la seguridad del sistema. Sin monotonía estricta no hay función inversa, y sin función inversa no hay forma de descifrar el mensaje.
