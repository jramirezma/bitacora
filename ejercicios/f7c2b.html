Ejercicio

Si $a, b, c, d$ son reales positivos tales que $s = a+b+c+d$, demuestre que: $$(s-a)(s-b)(s-c)(s-d) \geq 81\,abcd.$$

Análisis

Cada factor $(s-a)$, $(s-b)$, $(s-c)$, $(s-d)$ es la suma de los otros tres. Por ejemplo $s - a = b+c+d$. Entonces el producto es $(b+c+d)(a+c+d)(a+b+d)(a+b+c)$, y a cada factor de tres sumandos positivos le aplicamos AM-GM para tres términos: $u+v+w \geq 3\sqrt[3]{uvw}$.

$s - a = b+c+d \geq 3\sqrt[3]{bcd}$. Aplicando esto a los cuatro factores y multiplicando, el producto de las raíces cúbicas colapsa en $abcd$.

Solución

Desarrollo
Justificación
$$s - a = b+c+d$$ $$s - b = a+c+d$$ $$s - c = a+b+d$$ $$s - d = a+b+c$$
Sustitución de $s$
Como $s = a+b+c+d$, cada resto es simplemente la suma de los otros tres. Todos son positivos porque $a,b,c,d > 0$.
$$b+c+d \geq 3\sqrt[3]{bcd}$$ $$a+c+d \geq 3\sqrt[3]{acd}$$ $$a+b+d \geq 3\sqrt[3]{abd}$$ $$a+b+c \geq 3\sqrt[3]{abc}$$
AM-GM para tres términos
Para $u,v,w > 0$: $u+v+w \geq 3\sqrt[3]{uvw}$. Se obtiene aplicando AM-GM dos veces: primero $u+v \geq 2\sqrt{uv}$, luego al resultado con $w$. Aplicamos a cada factor con los tres positivos correspondientes.
$$(s-a)(s-b)(s-c)(s-d)$$ $$\geq 3\sqrt[3]{bcd}\cdot 3\sqrt[3]{acd}\cdot 3\sqrt[3]{abd}\cdot 3\sqrt[3]{abc}$$ $$= 81\sqrt[3]{bcd}\cdot\sqrt[3]{acd}\cdot\sqrt[3]{abd}\cdot\sqrt[3]{abc}$$
Producto de desigualdades
Multiplicamos las cuatro desigualdades. Válido porque todos los miembros son positivos. El coeficiente $3^4 = 81$ aparece al frente.
$$\sqrt[3]{bcd}\cdot\sqrt[3]{acd}\cdot\sqrt[3]{abd}\cdot\sqrt[3]{abc}$$ $$= \sqrt[3]{a^3b^3c^3d^3} = abcd$$
Simplificación del producto de raíces
Cada variable aparece exactamente tres veces entre los cuatro productos: $a$ aparece en $acd$, $abd$, $abc$; lo mismo para $b$, $c$, $d$. Entonces el producto bajo la raíz es $a^3b^3c^3d^3$, y $\sqrt[3]{a^3b^3c^3d^3} = abcd$.
$$\boxed{(s-a)(s-b)(s-c)(s-d) \geq 81\,abcd}$$
Conclusión
Encadenando. La igualdad se alcanza cuando $a = b = c = d$, es decir $s = 4a$.
Resultado. Para $a,b,c,d > 0$ con $s = a+b+c+d$, aplicando AM-GM de tres términos a cada factor $(s-x) = $ suma de los otros tres: $$\boxed{(s-a)(s-b)(s-c)(s-d) \geq 81\,abcd}$$ La igualdad se alcanza si y solo si $a = b = c = d$.
