Enunciado

Demuestre la ley de cancelación para la adición en los números reales: si $a + b = a + c$, entonces $b = c$.

Solución

Hipótesis: $a + b = a + c$ para algunos números reales $a, b, c$.

Tesis: $b = c$.

Estrategia: Usar la existencia del inverso aditivo $-a$ y la propiedad asociativa.

DesarrolloJustificación
$$a + b = a + c$$ Hipótesis
$$(a + b) + (-a) = (a + c) + (-a)$$ Suma $-a$ a ambos miembros
$$a + b + (-a) = a + c + (-a)$$ Asociatividad de la suma
$$(a + (-a)) + b = (a + (-a)) + c$$ Asociatividad de la suma
$$0 + b = 0 + c$$ Definición de inverso aditivo
$$b = c$$ Propiedad identidad aditiva
