El problema

Sea $f(x) = ax + b$, y suponga que para todo $x \in \mathbb{R}$ se tiene que $f(f(x)) = bx + a$. Calcule el valor de $a + b$.

¿De qué se trata esto?

Tenés una función lineal $f(x) = ax + b$ con dos constantes desconocidas. La condición $f(f(x)) = bx + a$ te dice cómo queda la función cuando se aplica dos veces seguidas. La estrategia es calcular $f(f(x))$ explícitamente y, como ese resultado tiene que coincidir con $bx + a$ para todo $x$, los coeficientes de ambos lados deben ser exactamente iguales.

Dos polinomios son iguales para todo $x$ si y solo si tienen los mismos coeficientes término a término. Eso convierte una igualdad de funciones en un sistema de dos ecuaciones con $a$ y $b$.

Este principio se llama identificación de coeficientes: en lugar de resolver una ecuación con valores numéricos, igualás el coeficiente de $x$ con el coeficiente de $x$, y el término independiente con el término independiente. De ahí salen las dos ecuaciones del sistema.

La función se aplica a sí misma y el resultado ya está dado. Solo hay que calcular ese resultado y ver qué condiciones impone.

Por si te lo preguntabas

La composición de una función consigo misma se llama iteración, y aparece en lugares inesperados. Los fractales más conocidos se construyen aplicando una función simple una y otra vez, miles de veces. En ese contexto, la pregunta natural no es "¿cuánto vale $f(f(x))$?" sino "¿hacia dónde converge la secuencia $x, f(x), f(f(x)), \ldots$?". La respuesta depende de los coeficientes de $f$, exactamente el tipo de información que se extrae con identificación de coeficientes.

Vamos paso a paso

Paso 1 — calcular $f(f(x))$

Desarrollo
Justificación
$f(x) = ax + b$
Función dada.
$f(f(x)) = f(ax + b)$
Se sustituye $f(x)$ como argumento de $f$.
$= a(ax+b) + b$
Se aplica la regla de $f$: multiplicar el argumento por $a$ y sumar $b$.
$= a^2x + ab + b$
Se distribuye $a$ y se agrupan los términos independientes.

Paso 2 — identificar coeficientes

Desarrollo
Justificación
$a^2x + ab + b = bx + a$
Se iguala el resultado con la condición del enunciado.
$a^2 = b$
Coeficientes de $x$: deben coincidir en ambos lados.
$ab + b = a$
Términos independientes: deben coincidir en ambos lados.

Paso 3 — resolver el sistema

Desarrollo
Justificación
$b(a + 1) = a$
Se factoriza $b$ en la segunda ecuación.
$a^2(a+1) = a$
Se sustituye $b = a^2$ de la primera ecuación.
$a^3 + a^2 - a = 0$
Se expande y se pasan todos los términos al mismo lado.
$a(a^2 + a - 1) = 0$
Se factoriza $a$.
$a = 0 \qquad$ o $\qquad a = \dfrac{-1 \pm \sqrt{5}}{2}$
Dos casos: $a = 0$ directo, o fórmula cuadrática aplicada a $a^2 + a - 1 = 0$.

Paso 4 — calcular $a + b$ en cada caso

Desarrollo
Justificación
$a = 0 \Rightarrow b = 0^2 = 0$
$a + b = 0$
Caso trivial: $f(x) = 0$, que cumple $f(f(x)) = 0 = bx + a$.
$a = \dfrac{-1+\sqrt{5}}{2}$

$b = a^2 = \dfrac{(-1+\sqrt{5})^2}{4} = \dfrac{6-2\sqrt{5}}{4} = \dfrac{3-\sqrt{5}}{2}$
Se eleva $a$ al cuadrado con $(p+q)^2 = p^2+2pq+q^2$ y se simplifica.
$a + b = \dfrac{-1+\sqrt{5}}{2} + \dfrac{3-\sqrt{5}}{2} = \dfrac{2}{2} = 1$
Los $\sqrt{5}$ se cancelan entre sí.
$a = \dfrac{-1-\sqrt{5}}{2}$

$b = a^2 = \dfrac{(-1-\sqrt{5})^2}{4} = \dfrac{6+2\sqrt{5}}{4} = \dfrac{3+\sqrt{5}}{2}$
Mismo procedimiento con la otra raíz.
$a + b = \dfrac{-1-\sqrt{5}}{2} + \dfrac{3+\sqrt{5}}{2} = \dfrac{2}{2} = 1$
Los $\sqrt{5}$ también se cancelan. Mismo resultado que el caso anterior.

Una prueba rápida

Para $a = \dfrac{-1+\sqrt{5}}{2}$ y $b = \dfrac{3-\sqrt{5}}{2}$, se verifica que $a^2 = b$:

$$a^2 = \left(\frac{-1+\sqrt{5}}{2}\right)^2 = \frac{6 - 2\sqrt{5}}{4} = \frac{3-\sqrt{5}}{2} = b \checkmark$$

Y que $ab + b = a$:

$$b(a+1) = \frac{3-\sqrt{5}}{2} \cdot \frac{1+\sqrt{5}}{2} = \frac{3+3\sqrt{5}-\sqrt{5}-5}{4} = \frac{-2+2\sqrt{5}}{4} = \frac{-1+\sqrt{5}}{2} = a \checkmark$$
Resultado. Los valores posibles de $a + b$ son: $$\boxed{a + b = 0} \quad \text{(caso trivial)} \qquad \text{y} \qquad \boxed{a + b = 1} \quad \text{(casos no triviales)}$$

¿Dónde ya existe esto?

Identificar coeficientes es exactamente lo que hace un compilador cuando resuelve un sistema de restricciones de tipos. Cuando el compilador detecta que dos expresiones deben ser del mismo tipo para todo valor posible, no compara caso por caso: compara las estructuras, igual que acá se comparan el coeficiente de $x$ y el término independiente por separado. En lenguajes con inferencia de tipos, como Haskell o Rust, el compilador construye ecuaciones sobre los tipos desconocidos y las resuelve sin que el programador tenga que escribir los tipos explícitamente. La matemática detrás de esa inferencia es la misma idea aplicada en este ejercicio.

Si quieres ir más lejos

  1. En este ejercicio, $f(f(x)) = bx + a$ tiene la misma forma que $f(x) = ax + b$ pero con $a$ y $b$ intercambiados. ¿Qué pasa si en cambio se pide que $f(f(x)) = f(x)$ para todo $x$? ¿Qué condición impone eso sobre $a$ y $b$?
  2. La identificación de coeficientes dice que si dos polinomios son iguales para todo $x$, sus coeficientes coinciden. ¿Por qué esa afirmación es verdadera? ¿Qué pasa si solo se sabe que son iguales para un valor particular de $x$?
  3. En la resolución del sistema, se obtiene la ecuación $a(a^2 + a - 1) = 0$. ¿Qué significa geométricamente que $a = 0$ sea una solución? ¿Cómo se ve la función $f(x) = 0x + 0$?
  4. Si $f(x) = ax + b$ y se define $g = f \circ f \circ f$ (la función aplicada tres veces), ¿cuál sería la forma general de $g(x)$? ¿Qué coeficientes tendría en función de $a$ y $b$?
  5. El número $\frac{-1+\sqrt{5}}{2}$ que aparece como solución es la razón áurea menos 1. ¿Tiene alguna propiedad especial que explique por qué aparece naturalmente en este tipo de ecuaciones con autorreferencia?
