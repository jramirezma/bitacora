Ejercicio

Diga si las siguientes afirmaciones son ciertas o falsas:

a) $\inf\mathbb{N} = 1$.

b) $\sup[1;3) = 3$.

c) $\inf(\mathbb{N}\cap(1;6)) = 2$.

d) $\sup\mathbb{N} = +\infty$.

Análisis

Recordamos las definiciones. El ínfimo de un conjunto $S$ es la mayor de sus cotas inferiores. El supremo es la menor de sus cotas superiores. Si el conjunto no está acotado superiormente, se dice que su supremo es $+\infty$ (con la convención de la recta extendida).

El ínfimo pertenece al conjunto si y solo si es el mínimo. El supremo pertenece al conjunto si y solo si es el máximo. $\mathbb{N} = \{1, 2, 3, \ldots\}$ no tiene cota superior en $\mathbb{R}$.

Solución

Desarrollo
Justificación

a) $\inf\mathbb{N} = 1$.

$$\mathbb{N} = \{1,2,3,\ldots\}$$ $$1 \in \mathbb{N}, \quad n \geq 1 \text{ para todo } n\in\mathbb{N}$$ $$\Rightarrow \inf\mathbb{N} = \min\mathbb{N} = 1$$ $$\textbf{VERDADERO}$$
Ínfimo de $\mathbb{N}$
$1$ es cota inferior de $\mathbb{N}$ porque $n \geq 1$ para todo natural. Además $1 \in \mathbb{N}$, así que es la mayor cota inferior, es decir el ínfimo. Y como pertenece al conjunto, es también el mínimo.

b) $\sup[1;3) = 3$.

$$[1,3) = \{x\in\mathbb{R}: 1\leq x < 3\}$$ $$x < 3 \text{ para todo } x\in[1,3)$$ $$\text{Para todo } \varepsilon>0:\; 3-\varepsilon \in [1,3) \text{ si } \varepsilon < 2$$ $$\Rightarrow \sup[1,3) = 3$$ $$\textbf{VERDADERO}$$
Supremo de $[1,3)$
$3$ es cota superior porque $x < 3$ para todo $x \in [1,3)$. Es la menor cota superior: cualquier $M < 3$ no es cota superior porque $\max(1, M) < 3$ y podemos encontrar elementos del conjunto entre $M$ y $3$. Como $3 \notin [1,3)$, no es máximo.

c) $\inf(\mathbb{N}\cap(1;6)) = 2$.

$$\mathbb{N}\cap(1,6) = \{2,3,4,5\}$$ $$\min\{2,3,4,5\} = 2$$ $$\Rightarrow \inf(\mathbb{N}\cap(1,6)) = 2$$ $$\textbf{VERDADERO}$$
Ínfimo de $\mathbb{N}\cap(1,6)$
El intervalo $(1,6)$ contiene los naturales $2,3,4,5$. El conjunto es finito, así que tiene mínimo: $2$. El ínfimo de un conjunto finito es su mínimo.

d) $\sup\mathbb{N} = +\infty$.

$$\text{Para todo } M\in\mathbb{R},\; \exists\, n\in\mathbb{N}: n > M$$ $$\Rightarrow \mathbb{N} \text{ no está acotado superiormente}$$ $$\Rightarrow \sup\mathbb{N} = +\infty$$ $$\textbf{VERDADERO}$$
Supremo de $\mathbb{N}$
Por el principio arquimediano, para todo real $M$ existe un natural mayor que $M$. Entonces $\mathbb{N}$ no tiene cota superior en $\mathbb{R}$. Con la convención de la recta extendida $\overline{\mathbb{R}}$, se escribe $\sup\mathbb{N} = +\infty$.
Resultado. Las cuatro afirmaciones son verdaderas:

a) $\inf\mathbb{N} = \min\mathbb{N} = 1$. ✓
b) $\sup[1,3) = 3$ (no es máximo pues $3\notin[1,3)$). ✓
c) $\mathbb{N}\cap(1,6) = \{2,3,4,5\}$, $\inf = \min = 2$. ✓
d) $\mathbb{N}$ no está acotado superiormente, $\sup\mathbb{N} = +\infty$. ✓
