El problema

Sea $$f(x) = \begin{cases} ax + b, & \text{si } x < 1; \\[4pt] \dfrac{1}{ax - b}, & \text{si } x > 1. \end{cases}$$ Encuentre $a$ y $b$ de tal forma que $\displaystyle\lim_{x \to 1^-} f(x) = 1$ y $\displaystyle\lim_{x \to 1^+} f(x) = \dfrac{1}{2}$.

¿De qué se trata esto?

Cada condición sobre un límite lateral se traduce en una ecuación con $a$ y $b$. El límite por izquierda usa el primer trozo; el límite por derecha usa el segundo. Calcular cada límite lateral e igualarlo al valor pedido da un sistema de dos ecuaciones con dos incógnitas.

Límite por izquierda: $\lim_{x \to 1^-}(ax + b) = a \cdot 1 + b = a + b$. Esto debe ser igual a $1$, así que $a + b = 1$.

Límite por derecha: $\lim_{x \to 1^+}\dfrac{1}{ax - b} = \dfrac{1}{a \cdot 1 - b} = \dfrac{1}{a - b}$. Esto debe ser igual a $\dfrac{1}{2}$, así que $a - b = 2$.

Con las dos ecuaciones planteadas, resolver el sistema es aritmética directa. Sumar las ecuaciones elimina $b$; restar las ecuaciones elimina $a$.

Un ejercicio de límites que termina siendo un sistema $2\times 2$. Los límites hicieron el trabajo difícil — plantear las ecuaciones — y la aritmética cierra el asunto.

Por si te lo preguntabas

Imponer condiciones sobre límites laterales para determinar parámetros de una función es exactamente lo que hacen los algoritmos de interpolación por splines. Dado un conjunto de puntos por los que debe pasar una curva, el algoritmo plantea un sistema de ecuaciones donde cada condición — pasar por el punto, tener cierta pendiente, ser continua en las costuras — se traduce en una ecuación lineal. El resultado es un sistema que puede tener decenas o cientos de ecuaciones, pero cuya estructura es siempre la misma: cada condición impuesta es una ecuación, y los parámetros desconocidos son las incógnitas. Lo que en este ejercicio son dos ecuaciones con $a$ y $b$, en un spline cúbico con $n$ segmentos son $4n$ ecuaciones con $4n$ coeficientes.

Vamos paso a paso

Condición 1: límite por izquierda

Desarrollo
Justificación
$\lim_{x \to 1^-} f(x) = \lim_{x \to 1^-}(ax + b)$
Para $x < 1$ rige el primer trozo.
$= a(1) + b = a + b$
Polinomio continuo; sustitución directa con $x = 1$.
$a + b = 1$
Se iguala al valor pedido. — Ecuación (I)

Condición 2: límite por derecha

Desarrollo
Justificación
$\lim_{x \to 1^+} f(x) = \lim_{x \to 1^+} \dfrac{1}{ax - b}$
Para $x > 1$ rige el segundo trozo.
$= \dfrac{1}{a(1) - b} = \dfrac{1}{a - b}$
Función racional continua en $x = 1$ si $a - b \neq 0$; sustitución directa.
$\dfrac{1}{a - b} = \dfrac{1}{2}$
Se iguala al valor pedido.
$a - b = 2$
Invertir ambos lados (ambos positivos). — Ecuación (II)

Resolución del sistema

Desarrollo
Justificación
$\begin{cases} a + b = 1 \\ a - b = 2 \end{cases}$
Sistema $2 \times 2$ con incógnitas $a$ y $b$.
$(a + b) + (a - b) = 1 + 2$
Se suman las dos ecuaciones para eliminar $b$.
$2a = 3 \;\Rightarrow\; a = \dfrac{3}{2}$
Aritmética.
$b = 1 - a = 1 - \dfrac{3}{2} = -\dfrac{1}{2}$
Se sustituye $a$ en la ecuación (I).

Chequeemos que esto tiene sentido

Con $a = 3/2$ y $b = -1/2$:

Límite por izquierda: $\lim_{x \to 1^-}\!\left(\tfrac{3}{2}x - \tfrac{1}{2}\right) = \tfrac{3}{2} - \tfrac{1}{2} = 1$. ✓

Límite por derecha: $\lim_{x \to 1^+}\dfrac{1}{\frac{3}{2}x + \frac{1}{2}} = \dfrac{1}{\frac{3}{2} + \frac{1}{2}} = \dfrac{1}{2}$. ✓

También se verifica que el denominador del segundo trozo no se anula en $x = 1$: $a - b = 3/2 + 1/2 = 2 \neq 0$. ✓

Resultado. $$a = \frac{3}{2}, \qquad b = -\frac{1}{2}.$$ Con estos valores: $\displaystyle\lim_{x \to 1^-} f(x) = 1$ y $\displaystyle\lim_{x \to 1^+} f(x) = \frac{1}{2}$.

¿Dónde ya existe esto?

En diseño de fuentes tipográficas, cada carácter está compuesto por segmentos de curva definidos por parámetros. Para que las curvas sean suaves en los puntos de unión, se imponen condiciones sobre los límites laterales de la posición y la pendiente — exactamente como en este ejercicio, pero con más parámetros. El proceso de encontrar los coeficientes de cada segmento que satisfagan todas esas condiciones simultáneamente es la resolución de un sistema lineal. Los diseñadores de fuentes ajustan los puntos de control visualmente, y el software resuelve el sistema en tiempo real para actualizar los parámetros de cada curva. Lo que aquí son dos ecuaciones con dos incógnitas, en una fuente tipográfica completa puede ser un sistema de miles de ecuaciones resuelto en milisegundos.
