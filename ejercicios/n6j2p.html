Ejercicio

Resuelva:

a) $x^3 < a^4 x$.     b) $\dfrac{x^2 - a^4}{x^2 - 4a^4} > 0$.

Análisis

El parámetro $a$ es un número real fijo. La solución depende del valor de $a$, así que hay que abrir casos. La estrategia es la de siempre: pasar todo al mismo lado, factorizar, y analizar signos. Pero ahora los puntos críticos dependen de $a$, y según el signo o magnitud de $a$ los intervalos cambian.

Notar que $a^4 \geq 0$ siempre. Si $a = 0$ la inecuación degenera. Para $a \neq 0$, $a^4 > 0$ y aparecen raíces reales no triviales.

Solución — parte a) $\ x^3 < a^4 x$

Desarrollo
Justificación
$$x^3 - a^4 x < 0$$ $$x(x^2 - a^4) < 0$$ $$x(x-a^2)(x+a^2) < 0$$
Pasar al mismo lado y factorizar
Factorizamos $x$ y luego $x^2 - a^4 = (x-a^2)(x+a^2)$. Como $a^4 \geq 0$, las raíces son $x = 0$, $x = a^2$, $x = -a^2$.

Caso $a = 0$:

$$x \cdot x \cdot x = x^3 < 0 \;\Rightarrow\; x < 0$$
Caso $a = 0$
Las tres raíces colapsan en $0$. El producto es $x^3$, negativo solo para $x < 0$.

Caso $a \neq 0$: $a^2 > 0$, raíces $-a^2 < 0 < a^2$.

Análisis de $x(x+a^2)(x-a^2)$:

  • $x < -a^2$: $(-)(-)(-) = -$ ✓
  • $-a^2 < x < 0$: $(-)(+)(-) = +$ ✗
  • $0 < x < a^2$: $(+)(+)(-) = -$ ✓
  • $x > a^2$: $(+)(+)(+) = +$ ✗
Análisis de signos para $a \neq 0$
Tres raíces distintas dividen la recta en cuatro intervalos. Evaluamos el signo del producto en cada uno. Los extremos se excluyen (desigualdad estricta).
$$\boxed{x \in (-\infty, -a^2) \cup (0, a^2)} \quad (a \neq 0)$$ $$\boxed{x \in (-\infty, 0)} \quad (a = 0)$$
Solución
Los intervalos donde el producto es negativo.

Solución — parte b) $\ \dfrac{x^2 - a^4}{x^2 - 4a^4} > 0$

Desarrollo
Justificación
$$\frac{(x-a^2)(x+a^2)}{(x-2a^2)(x+2a^2)} > 0$$
Factorización
$x^2 - a^4 = (x-a^2)(x+a^2)$ y $x^2 - 4a^4 = (x-2a^2)(x+2a^2)$. Puntos críticos: $\pm a^2$ (numerador) y $\pm 2a^2$ (denominador).

Caso $a = 0$:

$$\frac{x^2}{x^2} = 1 > 0 \quad \text{para todo } x \neq 0$$
Caso $a = 0$
Numerador y denominador colapsan a $x^2$. El cociente es $1$ excepto en $x=0$.

Caso $a \neq 0$: orden de raíces:

$$-2a^2 < -a^2 < 0 < a^2 < 2a^2$$

Análisis de signos del cociente:

  • $x < -2a^2$: $\frac{(-)(-)}{(-)(-))} = +$ ✓
  • $-2a^2 < x < -a^2$: $\frac{(-)(-)}{(+)(-)} = -$ ✗
  • $-a^2 < x < a^2$: $\frac{(-)(+)}{(+)(-)} = +$ ✓
  • $a^2 < x < 2a^2$: $\frac{(+)(+)}{(+)(-)} = -$ ✗
  • $x > 2a^2$: $\frac{(+)(+)}{(+)(+)} = +$ ✓
Análisis de signos para $a \neq 0$
Cuatro puntos críticos dividen la recta en cinco intervalos. $\pm 2a^2$ excluidos (denominador nulo). $\pm a^2$ excluidos (desigualdad estricta, cociente $= 0$).
$$\boxed{x \in (-\infty,-2a^2)\cup(-a^2,a^2)\cup(2a^2,+\infty)} \quad (a\neq 0)$$ $$\boxed{x \in \mathbb{R}\setminus\{0\}} \quad (a = 0)$$
Solución
Intervalos donde el cociente es positivo. Todos los extremos excluidos.
Resultado.

a) $x^3 < a^4 x$: $$\text{Si } a=0:\ x\in(-\infty,0). \qquad \text{Si } a\neq 0:\ x\in(-\infty,-a^2)\cup(0,a^2).$$ b) $\dfrac{x^2-a^4}{x^2-4a^4}>0$: $$\text{Si } a=0:\ x\in\mathbb{R}\setminus\{0\}. \qquad \text{Si } a\neq 0:\ x\in(-\infty,-2a^2)\cup(-a^2,a^2)\cup(2a^2,+\infty).$$
