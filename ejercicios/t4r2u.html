Ejercicio

Resuelva:

a) $x^2 - 2|x| - 3 \geq 0$.

b) $|2x+1| \geq 2+x$.

c) $|x^2-4| < -2x+4$.

d) $3 + |x^2+x| < \dfrac{15}{4}$.

Análisis

La estrategia general es abrir casos según el signo del argumento del valor absoluto. Una vez eliminado el valor absoluto en cada caso, se resuelve la inecuación resultante y se intersecta con la condición del caso. Para las partes simétricas en $x$ (como a), conviene sustituir $t = |x| \geq 0$.

$|E| \geq F$ se cumple cuando $E \geq F$ o $E \leq -F$. $|E| < F$ (con $F > 0$) equivale a $-F < E < F$. Estas equivalencias evitan abrir casos manualmente.

Solución — a) $\ x^2 - 2|x| - 3 \geq 0$

Desarrollo
Justificación
$$t = |x| \geq 0$$ $$t^2 - 2t - 3 \geq 0$$ $$(t-3)(t+1) \geq 0$$
Sustitución $t = |x|$
Como $x^2 = |x|^2 = t^2$, la expresión se convierte en cuadrática en $t$. Factorizamos: raíces $t = 3$ y $t = -1$.
$$t \leq -1 \quad \text{o} \quad t \geq 3$$ $$\text{Como } t \geq 0: \quad t \geq 3$$
Solución en $t$
El producto es $\geq 0$ fuera de las raíces. $t \leq -1$ es imposible con $t \geq 0$.
$$|x| \geq 3 \;\Leftrightarrow\; x \leq -3 \text{ o } x \geq 3$$
Deshacer la sustitución
$|x| \geq 3$ significa que $x$ está a distancia $\geq 3$ del origen.
$$\boxed{x \in (-\infty, -3] \cup [3, +\infty)}$$
Solución
Los extremos se incluyen por la desigualdad no estricta.

Solución — b) $\ |2x+1| \geq 2+x$

Desarrollo
Justificación

Caso 1: $2x+1 \geq 0$, es decir $x \geq -\frac{1}{2}$.

$$2x+1 \geq 2+x \;\Rightarrow\; x \geq 1$$ $$\text{Intersección: } x \geq 1$$
Caso $2x+1 \geq 0$
El valor absoluto no cambia el signo. Despejamos y tomamos la intersección con la condición del caso.

Caso 2: $2x+1 < 0$, es decir $x < -\frac{1}{2}$.

$$-(2x+1) \geq 2+x \;\Rightarrow\; -2x-1 \geq 2+x \;\Rightarrow\; -3x \geq 3 \;\Rightarrow\; x \leq -1$$ $$\text{Intersección: } x \leq -1$$
Caso $2x+1 < 0$
El valor absoluto invierte el signo. Despejamos (dividir por $-3$ invierte la desigualdad) y tomamos la intersección con $x < -\frac{1}{2}$.
$$\boxed{x \in (-\infty, -1] \cup [1, +\infty)}$$
Unión de los dos casos
Reunimos ambas soluciones. Los extremos se incluyen.

Solución — c) $\ |x^2-4| < -2x+4$

Desarrollo
Justificación
$$-2x+4 > 0 \;\Rightarrow\; x < 2$$
Condición necesaria
El lado derecho debe ser positivo para que $|E| < F$ tenga solución.
$$-(- 2x+4) < x^2-4 < -2x+4$$ $$2x-4 < x^2-4 < -2x+4$$
Equivalencia de $|E| < F$
$|E| < F$ con $F > 0$ equivale a $-F < E < F$.

Desigualdad izquierda: $2x-4 < x^2-4$

$$0 < x^2-2x = x(x-2)$$ $$x < 0 \text{ o } x > 2$$
Primera inecuación
$x(x-2) > 0$: producto positivo fuera de las raíces $0$ y $2$.

Desigualdad derecha: $x^2-4 < -2x+4$

$$x^2+2x-8 < 0 \;\Rightarrow\; (x+4)(x-2) < 0$$ $$-4 < x < 2$$
Segunda inecuación
Factorizamos y aplicamos análisis de signos. Producto negativo entre las raíces.
$$(x < 0 \text{ o } x > 2) \cap (-4 < x < 2) \cap (x < 2)$$ $$= -4 < x < 0$$
Intersección
Cruzamos las tres condiciones. $x > 2$ queda eliminado por $x < 2$. De $x < 0$ intersectado con $-4 < x < 2$ queda $-4 < x < 0$.
$$\boxed{x \in (-4, 0)}$$
Solución
Extremos excluidos por desigualdades estrictas.

Solución — d) $\ 3 + |x^2+x| < \dfrac{15}{4}$

Desarrollo
Justificación
$$|x^2+x| < \frac{15}{4} - 3 = \frac{3}{4}$$
Despeje
Restamos $3$ a ambos lados.
$$-\frac{3}{4} < x^2+x < \frac{3}{4}$$
Equivalencia de $|E| < F$
Con $F = \frac{3}{4} > 0$.

Desigualdad derecha: $x^2+x < \frac{3}{4}$

$$x^2+x-\frac{3}{4} < 0 \;\Rightarrow\; \left(x+\frac{3}{2}\right)\!\left(x-\frac{1}{2}\right) < 0$$ $$-\frac{3}{2} < x < \frac{1}{2}$$
Primera inecuación
Raíces de $x^2+x-\frac{3}{4}=0$: $x = \frac{-1\pm\sqrt{1+3}}{2} = \frac{-1\pm 2}{2}$, es decir $x = \frac{1}{2}$ y $x = -\frac{3}{2}$.

Desigualdad izquierda: $x^2+x > -\frac{3}{4}$

$$x^2+x+\frac{3}{4} > 0 \;\Rightarrow\; \left(x+\frac{1}{2}\right)^2+\frac{1}{2} > 0$$
Segunda inecuación
Completamos el cuadrado: $\left(x+\frac{1}{2}\right)^2 + \frac{1}{2} > 0$ es verdadero para todo $x \in \mathbb{R}$. No agrega restricción.
$$\boxed{x \in \left(-\tfrac{3}{2},\, \tfrac{1}{2}\right)}$$
Solución
Solo la primera inecuación restringe. Extremos excluidos.
Resultado.
a) $x \in (-\infty, -3] \cup [3, +\infty)$.
b) $x \in (-\infty, -1] \cup [1, +\infty)$.
c) $x \in (-4, 0)$.
d) $x \in \left(-\dfrac{3}{2},\, \dfrac{1}{2}\right)$.
