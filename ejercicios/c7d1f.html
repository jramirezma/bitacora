Ejercicio

Aplicando el ejercicio anterior, demuestre lo siguiente:

a) $a \cdot 0 = 0$.     b) $-0 = 0$.     c) $-(-a) = a$.     d) $(-1)a = -a$.

Análisis

Cada una de estas cuatro afirmaciones parece obvia, pero ninguna es gratuita: hay que construirlas desde los axiomas. La estrategia en todos los casos es la misma: fabricar una igualdad de la forma $a + X = a + Y$ y luego aplicar la ley de cancelación del ejercicio 1 para concluir $X = Y$.

Truco unificador: si logramos escribir $a + X = a + 0$ (o cualquier forma $a + X = a + Y$), la cancelación hace el trabajo y nos da $X = Y$ de inmediato.

Los axiomas disponibles son los del cuerpo: existencia de neutro aditivo ($a + 0 = a$), inverso aditivo ($a + (-a) = 0$), distributividad ($a(b+c) = ab + ac$) y todos los habituales. Nada más.

Solución — parte a) $\ a \cdot 0 = 0$

Desarrollo
Justificación
$a \cdot 0 = a \cdot (0 + 0) = a\cdot 0 + a\cdot 0$
Neutro aditivo + Distributividad
Primero: $0 = 0 + 0$ por el neutro aditivo. Luego: $a(0+0) = a\cdot 0 + a\cdot 0$ por distributividad. Encadenando, obtenemos $a\cdot 0 = a\cdot 0 + a\cdot 0$.
$a\cdot 0 + 0 = a\cdot 0 + a\cdot 0$
Neutro aditivo
El lado izquierdo de la igualdad anterior es $a\cdot 0$, que es lo mismo que $a\cdot 0 + 0$. Reescribimos así para tener exactamente la forma $x + A = x + B$ que pide la cancelación.
$\boxed{a \cdot 0 = 0}$
Ley de cancelación (ejercicio 1)
La igualdad $a\cdot 0 + 0 = a\cdot 0 + a\cdot 0$ tiene la forma $a\cdot 0 + A = a\cdot 0 + B$ con $A = 0$ y $B = a\cdot 0$. Cancelando $a\cdot 0$ concluimos $0 = a\cdot 0$.

Solución — parte b) $\ {-0} = 0$

Desarrollo
Justificación
$$0 + (-0) = 0$$
Axioma del inverso aditivo
Por definición, todo número $x$ satisface $x + (-x) = 0$. Con $x = 0$: $0 + (-0) = 0$.
$$0 + 0 = 0$$
Neutro aditivo
$0 + 0 = 0$ por el axioma del neutro. Entonces el lado derecho también puede escribirse así.
$$0 + (-0) = 0 + 0$$
Igualamos las dos expresiones
Ambos lados son iguales a $0$, así que son iguales entre sí.
$$\boxed{-0 = 0}$$
Ley de cancelación (ejercicio 1)
Cancelamos el $0$ del lado izquierdo en $0 + (-0) = 0 + 0$ y obtenemos $-0 = 0$.

Solución — parte c) $\ {-(-a)} = a$

Desarrollo
Justificación
$$(-a) + (-(-a)) = 0$$
Axioma del inverso aditivo
$-(-a)$ es el inverso aditivo de $(-a)$, así que $(-a) + (-(-a)) = 0$ por definición.
$$(-a) + a = 0$$
Axioma del inverso aditivo
También $(-a) + a = 0$, porque $a$ y $(-a)$ son inversos mutuos (el axioma es simétrico).
$$(-a) + (-(-a)) = (-a) + a$$
Igualamos
Ambas expresiones son iguales a $0$, así que son iguales entre sí.
$$\boxed{-(-a) = a}$$
Ley de cancelación (ejercicio 1)
Cancelamos $(-a)$ de ambos lados y concluimos.

Solución — parte d) $\ (-1)a = -a$

Desarrollo
Justificación
$$(-1)a + a = (-1)a + 1 \cdot a$$
Neutro multiplicativo
$1 \cdot a = a$ por el axioma del neutro multiplicativo. Reescribimos $a$ como $1\cdot a$.
$$(-1)a + 1\cdot a = (-1 + 1)\,a$$
Distributividad
$ba + ca = (b+c)a$. Factorizamos con $b = -1$, $c = 1$.
$$(-1 + 1)\,a = 0 \cdot a = 0$$
Inverso aditivo + parte a)
$-1 + 1 = 0$ por el axioma del inverso. Luego $0 \cdot a = 0$ por la parte a) de este ejercicio.
$$(-1)a + a = 0$$
Resumen de la cadena
Juntando los pasos: $(-1)a + a = 0$.
$$(-a) + a = 0$$
Axioma del inverso aditivo
Por definición, $(-a) + a = 0$.
$$(-1)a + a = (-a) + a$$
Igualamos
Ambas expresiones son iguales a $0$.
$$\boxed{(-1)a = -a}$$
Ley de cancelación (ejercicio 1)
Cancelamos $a$ de ambos lados.
Resultado. Usando la ley de cancelación para la adición junto con los axiomas de cuerpo, se demuestran las cuatro propiedades: $$\text{a)}\ a\cdot 0 = 0 \qquad \text{b)}\ {-0}=0 \qquad \text{c)}\ {-(-a)}=a \qquad \text{d)}\ (-1)a=-a$$ Todas siguen el mismo patrón: construir $x + A = x + B$ y cancelar $x$.
