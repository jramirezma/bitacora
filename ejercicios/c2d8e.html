El problema

Determine si la función es creciente o decreciente en su dominio: $$g(x) = x + 2^x + e^x$$

¿De qué se trata esto?

Esta función es una suma de tres partes: $x$, $2^x$ y $e^x$. El truco acá es que si cada una de esas partes es creciente por sí sola, entonces la suma también lo es. No hace falta analizar la expresión entera de un golpe: alcanza con ver que cada sumando crece cuando $x$ crece.

Si $f$ y $g$ son funciones crecientes, entonces $f + g$ también es creciente. La demostración lo usa directamente: la diferencia de la suma es la suma de las diferencias, y cada diferencia es negativa.

Las propiedades de $2^x$ y $e^x$ que necesitamos son: ambas son estrictamente positivas para todo $x$, y si $x_1 < x_2$ entonces $2^{x_1} < 2^{x_2}$ y $e^{x_1} < e^{x_2}$. Eso es una propiedad conocida de las exponenciales con base mayor que 1.

Tres funciones crecientes se juntan. El resultado no podía ser otra cosa.

Ahora sí, a calcular

Desarrollo
Justificación
Sean $x_1, x_2 \in \mathbb{R}$ con $x_1 < x_2$
Se toman dos valores arbitrarios con $x_1 < x_2$. El dominio es todo $\mathbb{R}$.
$g(x_1) - g(x_2)$ $= (x_1 + 2^{x_1} + e^{x_1}) - (x_2 + 2^{x_2} + e^{x_2})$
Se sustituye la definición de $g$.
$= (x_1 - x_2) + (2^{x_1} - 2^{x_2}) + (e^{x_1} - e^{x_2})$
Se reagrupan los términos por tipo.
$x_1 - x_2 < 0$
Por hipótesis: $x_1 < x_2$.
$2^{x_1} - 2^{x_2} < 0$
La función $2^x$ es creciente (base $2 > 1$): $x_1 < x_2 \Rightarrow 2^{x_1} < 2^{x_2}$.
$e^{x_1} - e^{x_2} < 0$
La función $e^x$ es creciente (base $e > 1$): $x_1 < x_2 \Rightarrow e^{x_1} < e^{x_2}$.
$g(x_1) - g(x_2) < 0$
Suma de tres términos negativos: el resultado es negativo.
$g(x_1) < g(x_2)$
Cuando $x_1 < x_2$, el valor de $g$ sube. Por definición, $g$ es creciente.

¿El resultado aguanta?

Tomamos $x_1 = 0 < x_2 = 1$:

$g(0) = 0 + 2^0 + e^0 = 0 + 1 + 1 = 2$

$g(1) = 1 + 2^1 + e^1 = 1 + 2 + e \approx 1 + 2 + 2.718 = 5.718$

$g(0) = 2 < 5.718 \approx g(1)$ ✓

Tomamos también $x_1 = -2 < x_2 = -1$ para verificar en valores negativos:

$g(-2) = -2 + 2^{-2} + e^{-2} \approx -2 + 0.25 + 0.135 = -1.615$

$g(-1) = -1 + 2^{-1} + e^{-1} \approx -1 + 0.5 + 0.368 = -0.132$

$g(-2) \approx -1.615 < -0.132 \approx g(-1)$ ✓

Resultado. $g(x) = x + 2^x + e^x$ es $\boxed{\text{estrictamente creciente}}$ en todo $\mathbb{R}$.

Para todo $x_1, x_2 \in \mathbb{R}$ con $x_1 < x_2$ se tiene $g(x_1) < g(x_2)$.

¿Dónde ya existe esto?

La idea de que la suma de funciones crecientes es creciente aparece directamente en el diseño de funciones de costo en machine learning. Muchas funciones de pérdida se construyen sumando varios términos, cada uno penalizando un tipo distinto de error. Si cada término por separado crece cuando el modelo empeora, la suma también crece, lo que garantiza que minimizar la suma total equivale a mejorar en todos los frentes a la vez. Sin esa propiedad, podría pasar que mejorar en un término empeore otro sin que la suma lo refleje, y el entrenamiento perdería su dirección.
