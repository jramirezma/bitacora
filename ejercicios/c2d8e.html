El problema

Determine si la función es creciente o decreciente en su dominio: $$g(x) = x + 2^x + e^x$$

¿De qué se trata esto?

La función es una suma de tres partes: $x$, $2^x$ y $e^x$. La pregunta es si, cuando $x$ crece, $g(x)$ también crece siempre. La clave está en que no hace falta analizar la expresión entera de un golpe: si cada una de las tres partes crece por separado, la suma también crece.

Si $f$ y $h$ son funciones estrictamente crecientes, entonces $f + h$ también es estrictamente creciente. La demostración lo usa directamente: la diferencia de la suma es la suma de las diferencias, y cada diferencia resulta negativa.

Para las exponenciales se usa una propiedad fundamental: toda función $a^x$ con base $a > 1$ es estrictamente creciente. Tanto $2^x$ (base $2 > 1$) como $e^x$ (base $e \approx 2.718 > 1$) cumplen esa condición. Y la función identidad $x$ es creciente de forma obvia.

Tres funciones crecientes entran a una suma. El resultado no podía ser otra cosa.

Algo que vale la pena saber

La monotonía de una función —la propiedad de ser siempre creciente o siempre decreciente— garantiza algo valioso: la función tiene inversa. Si una función es estrictamente creciente, entonces a cada valor de salida le corresponde exactamente un valor de entrada, y eso es exactamente la condición para poder "deshacer" la función. En cambio, una función que sube y baja no tiene inversa global: el mismo valor de salida puede corresponder a dos entradas distintas, y ya no hay forma de saber cuál era la original.

Desarrollo
Justificación
Sean $x_1, x_2 \in \mathbb{R}$ con $x_1 < x_2$
Se toman dos valores arbitrarios del dominio. El dominio de $g$ es todo $\mathbb{R}$.
$g(x_1) - g(x_2)$ $= (x_1 + 2^{x_1} + e^{x_1}) - (x_2 + 2^{x_2} + e^{x_2})$
Se sustituye la definición de $g$ en cada término.
$= (x_1 - x_2) + (2^{x_1} - 2^{x_2}) + (e^{x_1} - e^{x_2})$
Se reagrupan los términos por tipo de función.
$x_1 - x_2 < 0$
Directo de la hipótesis: $x_1 < x_2$.
$2^{x_1} - 2^{x_2} < 0$
$2^x$ es creciente (base $2 > 1$): $x_1 < x_2 \Rightarrow 2^{x_1} < 2^{x_2}$.
$e^{x_1} - e^{x_2} < 0$
$e^x$ es creciente (base $e > 1$): $x_1 < x_2 \Rightarrow e^{x_1} < e^{x_2}$.
$g(x_1) - g(x_2) < 0$
Suma de tres cantidades negativas: el resultado es negativo.
$\therefore\; x_1 < x_2 \Rightarrow g(x_1) < g(x_2)$
Definición de función estrictamente creciente. Se cumple para todo par en $\mathbb{R}$.

¿El resultado aguanta?

Con $x_1 = 0 < x_2 = 1$:

$g(0) = 0 + 2^0 + e^0 = 0 + 1 + 1 = 2$

$g(1) = 1 + 2^1 + e^1 = 1 + 2 + e \approx 5.718$

$g(0) = 2 < 5.718 \approx g(1)$ ✓

Con $x_1 = -2 < x_2 = -1$ para verificar en valores negativos:

$g(-2) = -2 + 2^{-2} + e^{-2} \approx -2 + 0.25 + 0.135 = -1.615$

$g(-1) = -1 + 2^{-1} + e^{-1} \approx -1 + 0.5 + 0.368 = -0.132$

$g(-2) \approx -1.615 < -0.132 \approx g(-1)$ ✓

Resultado. $g(x) = x + 2^x + e^x$ es $\boxed{\text{estrictamente creciente}}$ en todo $\mathbb{R}$.

Para todo $x_1, x_2 \in \mathbb{R}$ con $x_1 < x_2$ se tiene $g(x_1) < g(x_2)$.

¿Dónde ya existe esto?

La propiedad de que la suma de funciones crecientes es creciente aparece directamente en el diseño de funciones de costo en machine learning. Muchas funciones de pérdida se construyen sumando varios términos, cada uno penalizando un tipo distinto de error. Si cada término crece cuando el modelo empeora, la suma también crece, lo que garantiza que minimizar el total equivale a mejorar en todos los frentes a la vez. Sin esa propiedad, podría ocurrir que mejorar en un término empeore otro sin que la suma lo refleje, y el proceso de entrenamiento perdería dirección.
