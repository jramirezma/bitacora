Ejercicio

Demuestre la ley de cancelación para la adición: si $a + b = a + c$, entonces $b = c$.

Análisis

La idea es sencilla: si dos números producen el mismo resultado al sumarle un tercero, entonces esos dos números deben ser iguales. Dicho así suena obvio, pero el punto es demostrarlo usando solo los axiomas de los números reales, sin dar nada por sentado.

La clave es que todo número real tiene un único inverso aditivo. Si sumamos ese inverso a ambos lados de la igualdad, el $a$ desaparece y queda $b = c$ solo.

Los axiomas que vamos a usar son exactamente tres: la existencia del inverso aditivo $(-a)$ tal que $a + (-a) = 0$; la existencia del neutro aditivo $0$ tal que $0 + x = x$; y la propiedad asociativa $(x + y) + z = x + (y + z)$. No hace falta nada más.

Demostrar algo "obvio" desde cero es como construir una silla partiendo de los átomos. Aburrido en apariencia, pero así se sabe que la silla no se va a romper.

Solución

Desarrollo
Justificación
$$a + b = a + c$$
Hipótesis
Es la premisa que nos dan. A partir de aquí todo debe seguirse solo con axiomas.
$$(-a) + (a + b) = (-a) + (a + c)$$
Suma del inverso aditivo de $a$
Sumamos $(-a)$ a ambos lados. Como la igualdad se preserva al aplicar la misma operación en ambos miembros, la igualdad sigue siendo válida.
$$((-a) + a) + b = ((-a) + a) + c$$
Axioma asociativo
Reagrupamos usando $(x + y) + z = x + (y + z)$. El paréntesis cambia de lugar, nada más.
$$0 + b = 0 + c$$
Axioma del inverso aditivo
Por definición, $(-a) + a = 0$. Cada par $(-a) + a$ colapsa a cero.
$$b = c$$
Axioma del neutro aditivo
Por definición, $0 + x = x$ para todo $x$. Entonces $0 + b = b$ y $0 + c = c$, y concluimos directamente.
Resultado. Partiendo únicamente de los axiomas de cuerpo (inverso aditivo, neutro aditivo y asociatividad), se concluye que si $a + b = a + c$, entonces $\boxed{b = c}$.

La cadena completa es: $$a+b=a+c \;\Rightarrow\; (-a)+(a+b)=(-a)+(a+c) \;\Rightarrow\; 0+b=0+c \;\Rightarrow\; b=c.$$
