Ejercicio

Demuestre lo siguiente:

a) Si $n > 1$, entonces $\dfrac{1}{n^3} < \dfrac{1}{n^2}$.

b) Si $n > 1$, entonces $\dfrac{2}{n^3} < \dfrac{3}{n^2}$.

c) Si $n \geq 8$, entonces $\dfrac{1}{\sqrt{2n+7}} \leq \dfrac{1}{\sqrt{18+n}}$.

Análisis

Las tres partes usan la misma lógica: para comparar dos fracciones con numeradores fijos, basta comparar sus denominadores, recordando que la fracción con denominador mayor es la menor. Y para comparar denominadores, se parte de la hipótesis sobre $n$ y se opera algebraicamente hasta llegar a la conclusión.

Si $0 < A < B$, entonces $\dfrac{1}{A} > \dfrac{1}{B}$: el orden se invierte al tomar recíprocos. Ese es el mecanismo central de las tres partes.

Solución — parte a)

Desarrollo
Justificación
$$n > 1$$
Hipótesis
Punto de partida.
$$n^2 > n \cdot 1 = n \quad \Rightarrow \quad n^3 > n^2$$
Multiplicación por $n^2 > 0$
Multiplicamos $n > 1$ por $n^2 > 0$. El sentido se conserva y obtenemos $n^3 > n^2$.
$$\boxed{\frac{1}{n^3} < \frac{1}{n^2}}$$
Recíprocos de positivos
Como $0 < n^2 < n^3$, al tomar recíprocos el orden se invierte.

Solución — parte b)

Desarrollo
Justificación
$$\frac{1}{n^3} < \frac{1}{n^2}$$
Parte a)
Ya demostrado para $n > 1$.
$$\frac{2}{n^3} < \frac{2}{n^2}$$
Multiplicación por $2 > 0$
Multiplicamos ambos lados por $2$. El sentido se conserva.
$$\frac{2}{n^2} < \frac{3}{n^2}$$
Comparación de numeradores
$2 < 3$ y el denominador $n^2 > 0$ es el mismo, así que $\frac{2}{n^2} < \frac{3}{n^2}$.
$$\boxed{\frac{2}{n^3} < \frac{3}{n^2}}$$
Transitividad
$\frac{2}{n^3} < \frac{2}{n^2} < \frac{3}{n^2}$, entonces $\frac{2}{n^3} < \frac{3}{n^2}$.

Solución — parte c)

Desarrollo
Justificación
$$n \geq 8$$
Hipótesis
Punto de partida.
$$n \geq 8 \;\Rightarrow\; n - 8 \geq 0 \;\Rightarrow\; 2n - 7 \geq n + 9$$ $$\;\Rightarrow\; 2n + 7 \geq n + 16 > n + 18 - 2$$
Manipulación algebraica
Buscamos comparar $2n+7$ con $n+18$. De $n \geq 8$ se sigue $n - 8 \geq 0$, es decir $n \geq 8$, y entonces $2n + 7 \geq n + 15$. Pero necesitamos $2n+7 \geq n+18$, es decir $n \geq 11$. Revisamos: para $n \geq 8$ esto no es siempre cierto. Comparamos directamente los radicandos.
$$2n+7 \leq n + 18 \iff n \leq 11$$
Revisión de la dirección
Para $8 \leq n \leq 11$: $2n+7 \leq n+18$. Para $n > 11$: $2n+7 > n+18$. La desigualdad del enunciado requiere $\sqrt{2n+7} \geq \sqrt{n+18}$, es decir $2n+7 \geq n+18$, es decir $n \geq 11$. Pero el enunciado dice $n \geq 8$. Verificamos $n=8$: $\sqrt{23} \leq \sqrt{26}$ ✓. Entonces la dirección correcta es $2n+7 \geq n+18$ para $n \geq 11$, y para $8 \leq n \leq 10$ verificar directamente.
$$n \geq 8 \;\Rightarrow\; n + 7 \geq 15 \;\Rightarrow\; 2n+7 \geq n+15$$ $$\text{y } n+18 \leq 2n+10 \text{ para } n \geq 8$$ $$\Rightarrow\; 2n+7 \geq n+18 \iff n \geq 11$$
Corrección: el enunciado dice $\sqrt{18+n}$, no $\sqrt{n+18}$
Leemos de nuevo: $\frac{1}{\sqrt{2n+7}} \leq \frac{1}{\sqrt{18+n}}$ equivale a $\sqrt{18+n} \leq \sqrt{2n+7}$, es decir $18+n \leq 2n+7$, es decir $11 \leq n$. Pero el enunciado asume $n \geq 8$. Para $n = 8$: $26 \leq 23$, falso. Releyendo el enunciado original: la desigualdad es $\frac{1}{\sqrt{2n+7}} \leq \frac{1}{\sqrt{18+n}}$.
$$\frac{1}{\sqrt{2n+7}} \leq \frac{1}{\sqrt{18+n}}$$ $$\iff \sqrt{18+n} \leq \sqrt{2n+7}$$ $$\iff 18+n \leq 2n+7$$ $$\iff 11 \leq n$$
Equivalencia de la desigualdad
Tomar recíprocos invierte el orden (ambos radicales positivos). Elevar al cuadrado preserva el orden (ambos lados positivos). La condición necesaria y suficiente es $n \geq 11$. Como $n \geq 8$ no implica $n \geq 11$, la desigualdad del enunciado es válida solo para $n \geq 11$. Para $n = 8, 9, 10$ la desigualdad es falsa: por ejemplo $n=8$ da $\frac{1}{\sqrt{23}} \leq \frac{1}{\sqrt{26}}$, es decir $\sqrt{26} \leq \sqrt{23}$, que es falso.
$$n \geq 11 \;\Rightarrow\; n - 11 \geq 0 \;\Rightarrow\; n+7 \geq 18$$ $$\;\Rightarrow\; 2n+7 \geq n+18$$ $$\;\Rightarrow\; \sqrt{2n+7} \geq \sqrt{n+18}$$ $$\;\Rightarrow\; \boxed{\frac{1}{\sqrt{2n+7}} \leq \frac{1}{\sqrt{n+18}}}$$
Demostración para $n \geq 11$ (que incluye $n \geq 8$ del enunciado con precaución)
La cadena es directa. La condición mínima real es $n \geq 11$, más fuerte que $n \geq 8$. Es posible que el enunciado tenga una errata o que $n$ sea entero y la condición $n \geq 8$ sea un error tipográfico por $n \geq 11$.
Resultado.
a) $n > 1 \Rightarrow n^3 > n^2 \Rightarrow \boxed{\dfrac{1}{n^3} < \dfrac{1}{n^2}}$.

b) Por transitividad: $\dfrac{2}{n^3} < \dfrac{2}{n^2} < \dfrac{3}{n^2}$, luego $\boxed{\dfrac{2}{n^3} < \dfrac{3}{n^2}}$.

c) La desigualdad $\dfrac{1}{\sqrt{2n+7}} \leq \dfrac{1}{\sqrt{18+n}}$ es equivalente a $n \geq 11$. La demostración válida es: $n \geq 11 \Rightarrow 2n+7 \geq n+18 \Rightarrow \sqrt{2n+7} \geq \sqrt{n+18} \Rightarrow \boxed{\dfrac{1}{\sqrt{2n+7}} \leq \dfrac{1}{\sqrt{n+18}}}$.
Nota: la condición $n \geq 8$ del enunciado parece ser una errata; la condición correcta es $n \geq 11$.
