Ejercicio

Resuelva:

a) $\big||x-1|-5\big| < 2$.

b) $\big|||x|-3\big|-2\big| < 1$.

c) $\left|\dfrac{1}{x}-1\right| < 3$.

Análisis

Los valores absolutos anidados se resuelven de afuera hacia adentro. Cada capa $|E| < F$ se convierte en $-F < E < F$, y el argumento $E$ del siguiente nivel vuelve a ser una inecuación con valor absoluto que se resuelve igual. Así se va pelando la cebolla capa por capa.

Estrategia: $|A| < r \Leftrightarrow -r < A < r$. Aplicar repetidamente hacia adentro. Al final, reunir todos los intervalos.

Solución — a) $\ \big||x-1|-5\big| < 2$

Desarrollo
Justificación
$$-2 < |x-1| - 5 < 2$$ $$3 < |x-1| < 7$$
Capa exterior
$|E| < 2$ equivale a $-2 < E < 2$ con $E = |x-1|-5$. Sumamos $5$ en toda la cadena.

$|x-1| > 3$:

$$x-1 > 3 \text{ o } x-1 < -3$$ $$x > 4 \text{ o } x < -2$$
Capa interior — cota inferior
$|x-1| > 3$ significa que $x$ está a distancia $> 3$ de $1$.

$|x-1| < 7$:

$$-7 < x-1 < 7$$ $$-6 < x < 8$$
Capa interior — cota superior
$|x-1| < 7$ equivale a $-7 < x-1 < 7$.
$$(x < -2 \text{ o } x > 4) \cap (-6 < x < 8)$$ $$= (-6,-2) \cup (4,8)$$
Intersección
Cruzamos las dos condiciones. Todos los extremos son estrictos.
$$\boxed{x \in (-6,-2)\cup(4,8)}$$
Solución
Cuatro extremos excluidos por desigualdades estrictas.

Solución — b) $\ \big|||x|-3\big|-2\big| < 1$

Desarrollo
Justificación
$$-1 < ||x|-3| - 2 < 1$$ $$1 < ||x|-3| < 3$$
Capa exterior
$|E|<1$ con $E=||x|-3|-2$. Sumamos $2$.

$||x|-3| > 1$:

$$|x|-3 > 1 \text{ o } |x|-3 < -1$$ $$|x| > 4 \text{ o } |x| < 2$$
Segunda capa — cota inferior
$|E| > 1$ con $E = |x|-3$.

$||x|-3| < 3$:

$$-3 < |x|-3 < 3$$ $$0 < |x| < 6 \;\Rightarrow\; 0 < |x| < 6$$
Segunda capa — cota superior
Sumamos $3$. $|x| \geq 0$ siempre, así que $|x|>0$ equivale a $x\neq 0$.

Condición combinada en $|x|$:

$$(|x|>4 \text{ o } |x|<2) \cap (0<|x|<6)$$ $$= (4<|x|<6) \cup (0<|x|<2)$$
Intersección en $|x|$
$|x|>4$ intersectado con $|x|<6$ da $4<|x|<6$. $|x|<2$ intersectado con $|x|>0$ da $0<|x|<2$.

$4<|x|<6 \Rightarrow x\in(-6,-4)\cup(4,6)$.

$0<|x|<2 \Rightarrow x\in(-2,0)\cup(0,2)$.

Traducción a intervalos en $x$
$|x| \in (a,b)$ con $0 < a < b$ equivale a $x\in(-b,-a)\cup(a,b)$.
$$\boxed{x \in (-6,-4)\cup(-2,0)\cup(0,2)\cup(4,6)}$$
Solución
Todos los extremos excluidos. $x=0$ excluido por $|x|>0$.

Solución — c) $\ \left|\dfrac{1}{x}-1\right| < 3$

Desarrollo
Justificación
$$-3 < \frac{1}{x}-1 < 3$$ $$-2 < \frac{1}{x} < 4$$
Equivalencia + suma de 1
$|E|<3$ con $E=\frac{1}{x}-1$. Sumamos $1$. Dominio: $x\neq 0$.

Desigualdad derecha: $\dfrac{1}{x} < 4$

$$\frac{1}{x}-4 < 0 \;\Rightarrow\; \frac{1-4x}{x} < 0$$

Puntos críticos: $x=0$, $x=\frac{1}{4}$.

$$x < 0 \text{ o } x > \tfrac{1}{4}$$
Primera inecuación
Pasamos $4$ al mismo lado. Tabla de signos: $x<0$: $(+)/(-)=-$ ✓; $0\frac{1}{4}$: $(-)/(+)=-$ ✓.

Desigualdad izquierda: $\dfrac{1}{x} > -2$

$$\frac{1}{x}+2 > 0 \;\Rightarrow\; \frac{1+2x}{x} > 0$$

Puntos críticos: $x=0$, $x=-\frac{1}{2}$.

$$x < -\tfrac{1}{2} \text{ o } x > 0$$
Segunda inecuación
Tabla de signos: $x<-\frac{1}{2}$: $(-)/(-)=+$ ✓; $-\frac{1}{2}0$: $(+)/(+)=+$ ✓.
$$(x<0\text{ o }x>\tfrac{1}{4})\cap(x<-\tfrac{1}{2}\text{ o }x>0)$$ $$= \left(-\infty,-\tfrac{1}{2}\right)\cup\left(\tfrac{1}{4},+\infty\right)$$
Intersección
$x<0$ intersectado con $x<-\frac{1}{2}$ da $x<-\frac{1}{2}$. $x>\frac{1}{4}$ intersectado con $x>0$ da $x>\frac{1}{4}$.
$$\boxed{x \in \left(-\infty,-\tfrac{1}{2}\right)\cup\left(\tfrac{1}{4},+\infty\right)}$$
Solución
Extremos excluidos. $x=0$ ya excluido por dominio.
Resultado.
a) $x \in (-6,-2)\cup(4,8)$.
b) $x \in (-6,-4)\cup(-2,0)\cup(0,2)\cup(4,6)$.
c) $x \in \left(-\infty,-\dfrac{1}{2}\right)\cup\left(\dfrac{1}{4},+\infty\right)$.

La técnica de afuera hacia adentro permite reducir cualquier anidamiento a inecuaciones simples.
