Ejercicio

Determine una asignación de valores de verdad para $A$, $B$, $C$, $D$ y $E$ que verifique que $$[(A \to (B \lor C)) \land (C \to (D \land E)) \land \neg D]$$ no implica tautológicamente a $A \to E$.

Análisis

Decir que una proposicion $\Phi$ implica tautologicamente a $\Psi$ significa que en toda fila donde $\Phi$ es verdadera, $\Psi$ tambien lo es. Escrito de otra forma: la implicacion $\Phi \to \Psi$ es una tautologia.

Para demostrar que $\Phi$ no implica tautologicamente a $\Psi$, basta encontrar una sola asignacion de valores donde $\Phi$ sea verdadera y $\Psi$ sea falsa al mismo tiempo. Ese es el contraejemplo que pide el ejercicio.

Llamemos $\Phi = (A \to (B \lor C)) \land (C \to (D \land E)) \land \neg D$ y $\Psi = A \to E$. Necesitamos que $\Phi$ sea verdadera y $\Psi$ sea falsa.

Para que $\Psi = A \to E$ sea falsa, necesitamos $A = V$ y $E = F$. Para que $\neg D$ sea verdadera, necesitamos $D = F$. Con $D = F$, la unica forma de que $C \to (D \land E)$ sea verdadera es que $C = F$. Con $C = F$ y $A = V$, necesitamos que $A \to (B \lor C) = V \to (B \lor F)$ sea verdadera, lo que exige $B = V$.

(Un solo contraejemplo basta para hundir una implicacion tautologica. La logica es implacable pero justa.)

Solución

Desarrollo
Justificación
Para que $\Psi = A \to E$ sea falsa:

$A = V$ y $E = F$
Condicion sobre Ψ
La implicacion $A \to E$ es falsa unicamente cuando $A$ es verdadera y $E$ es falsa. Esa es la unica combinacion que nos sirve para el contraejemplo.
Para que $\neg D$ sea verdadera:

$D = F$

Con $D = F$ y $E = F$:
$D \land E = F \land F = F$

Para que $C \to (D \land E) = C \to F$ sea verdadera:
$C = F$
Condicion sobre C y D
$\neg D$ verdadera obliga a $D = F$. Con $D$ y $E$ ambos falsos, $D \land E = F$. Para que $C \to F$ sea verdadera, el antecedente $C$ debe ser falso: una implicacion con consecuente falso solo se salva si el antecedente tambien es falso.
Con $A = V$ y $C = F$:
$B \lor C = B \lor F = B$

Para que $A \to (B \lor C) = V \to B$ sea verdadera:
$B = V$
Condicion sobre B
Con $A = V$, la implicacion $A \to (B \lor C)$ exige que el consecuente sea verdadero. Como $C = F$, el consecuente se reduce a $B$. Entonces $B$ debe ser verdadero.
Asignacion encontrada:

$A = V,\; B = V,\; C = F,\; D = F,\; E = F$

Verificacion:
SubformulaValor
$\neg D$V
$B \lor C$V
$A \to (B \lor C)$V
$D \land E$F
$C \to (D \land E)$V
$\Phi$ (conjuncion de las tres)V
$\Psi = A \to E$F
Verificacion del contraejemplo
La tabla confirma que con esta asignacion $\Phi$ es verdadera y $\Psi$ es falsa al mismo tiempo. Eso es exactamente lo que se necesita: una fila donde la premisa se cumple pero la conclusion no. Si $\Phi$ implicara tautologicamente a $\Psi$, esto seria imposible. Como ocurre, queda demostrado que la implicacion tautologica no se cumple.
Resultado. La asignacion $$A = V,\quad B = V,\quad C = F,\quad D = F,\quad E = F$$ es un contraejemplo: hace que $\Phi$ sea verdadera y $A \to E$ sea falsa simultaneamente, lo que prueba que $\Phi$ no implica tautologicamente a $A \to E$.
