Ejercicio

Resuelva:

a) $\left|\dfrac{x+2}{2x-3}\right| < 4$.    b) $\left|\dfrac{6-5x}{3+x}\right| \leq \dfrac{1}{2}$.    c) $\left|\dfrac{x+3}{x+16}\right| > \dfrac{3}{x-4}$.    d) $\left|\dfrac{x}{1+x}\right| \geq \dfrac{1}{x}$.

Análisis

La clave con $\left|\frac{A}{B}\right|$ es usar $\left|\frac{A}{B}\right| = \frac{|A|}{|B|}$ (ejercicio 19), lo que permite trabajar con numerador y denominador por separado. Alternativamente, $|E| < F$ (con $F > 0$) equivale a $-F < E < F$, que lleva a dos inecuaciones racionales estándar. En c) y d) el lado derecho depende de $x$, así que hay que analizar su signo antes de operar.

Nunca multiplicar por el denominador sin saber su signo. Siempre pasar todo al mismo lado y analizar el cociente resultante con tabla de signos.

Solución — a) $\ \left|\dfrac{x+2}{2x-3}\right| < 4$

Desarrollo
Justificación
$$-4 < \frac{x+2}{2x-3} < 4$$
Equivalencia $|E| < F$
Con $F = 4 > 0$. Dominio: $x \neq \frac{3}{2}$.

Desigualdad derecha: $\dfrac{x+2}{2x-3} - 4 < 0$

$$\frac{x+2 - 4(2x-3)}{2x-3} < 0 \;\Rightarrow\; \frac{-7x+14}{2x-3} < 0 \;\Rightarrow\; \frac{7(2-x)}{2x-3} < 0$$

Puntos críticos: $x=2$, $x=\frac{3}{2}$.

$$x < \tfrac{3}{2} \text{ o } x > 2$$
Primera inecuación
Pasamos $4$ al mismo lado y simplificamos. Tabla de signos: para $x < \frac{3}{2}$: $(+)/(-)= -$ ✓; para $\frac{3}{2}2$: $(-)/(+)=-$ ✓.

Desigualdad izquierda: $\dfrac{x+2}{2x-3} + 4 > 0$

$$\frac{x+2+4(2x-3)}{2x-3} > 0 \;\Rightarrow\; \frac{9x-10}{2x-3} > 0$$

Puntos críticos: $x=\frac{10}{9}$, $x=\frac{3}{2}$.

$$x < \tfrac{10}{9} \text{ o } x > \tfrac{3}{2}$$
Segunda inecuación
Tabla de signos: para $x<\frac{10}{9}$: $(-)/(-)=+$ ✓; para $\frac{10}{9}\frac{3}{2}$: $(+)/(+)=+$ ✓.
$$\left(x<\tfrac{3}{2}\text{ o }x>2\right)\cap\left(x<\tfrac{10}{9}\text{ o }x>\tfrac{3}{2}\right)$$ $$= \left(-\infty,\tfrac{10}{9}\right)\cup\left(\tfrac{3}{2},2\right)\cup\left(2,+\infty\right)$$ $$= \left(-\infty,\tfrac{10}{9}\right)\cup\left(\tfrac{3}{2},+\infty\right)\setminus\{2\}$$
Intersección
Cruzamos ambas condiciones por tramos. $x=2$ satisface ambas inecuaciones (cociente $= 0$ en la derecha y $> 0$ en la izquierda), verificamos: $\left|\frac{4}{1}\right| = 4$, no cumple la estricta. Lo excluimos.
$$\boxed{x \in \left(-\infty,\,\tfrac{10}{9}\right)\cup\left(\tfrac{3}{2},\,2\right)\cup\left(2,\,+\infty\right)}$$
Solución
$x = \frac{3}{2}$ excluido por dominio; $x=2$ excluido porque da igualdad, no desigualdad estricta.

Solución — b) $\ \left|\dfrac{6-5x}{3+x}\right| \leq \dfrac{1}{2}$

Desarrollo
Justificación
$$-\frac{1}{2} \leq \frac{6-5x}{3+x} \leq \frac{1}{2}$$
Equivalencia $|E| \leq F$
Con $F = \frac{1}{2} > 0$. Dominio: $x \neq -3$.

Desigualdad derecha: $\dfrac{6-5x}{3+x} - \dfrac{1}{2} \leq 0$

$$\frac{2(6-5x)-(3+x)}{2(3+x)} \leq 0 \;\Rightarrow\; \frac{9-11x}{2(3+x)} \leq 0$$

Puntos críticos: $x = \frac{9}{11}$, $x=-3$.

$$x \leq -3 \text{ (excluido) o } x \geq \tfrac{9}{11}$$ $$\Rightarrow x \in (-\infty,-3)\cup\left[\tfrac{9}{11},+\infty\right)$$
Primera inecuación
El cociente es $\leq 0$ cuando numerador y denominador tienen signos opuestos, o el numerador es $0$.

Desigualdad izquierda: $\dfrac{6-5x}{3+x} + \dfrac{1}{2} \geq 0$

$$\frac{2(6-5x)+(3+x)}{2(3+x)} \geq 0 \;\Rightarrow\; \frac{15-9x}{2(3+x)} \geq 0 \;\Rightarrow\; \frac{3(5-3x)}{2(3+x)} \geq 0$$

Puntos críticos: $x=\frac{5}{3}$, $x=-3$.

$$x \in (-3, \tfrac{5}{3}]\cup \text{ imposible para } x<-3$$ $$\Rightarrow x\in(-3,\tfrac{5}{3}]$$
Segunda inecuación
El cociente es $\geq 0$ cuando ambos tienen el mismo signo o el numerador es $0$. Para $x < -3$: numerador $+$, denominador $-$: cociente $-$ ✗. Para $-3 < x \leq \frac{5}{3}$: ambos $+$ ✓. Para $x > \frac{5}{3}$: num $-$, den $+$: $-$ ✗.
$$\left[(-\infty,-3)\cup[\tfrac{9}{11},+\infty)\right] \cap (-3,\tfrac{5}{3}]$$ $$= \left[\tfrac{9}{11},\,\tfrac{5}{3}\right]$$
Intersección
$(-\infty,-3) \cap (-3,\frac{5}{3}] = \emptyset$. $[\frac{9}{11},+\infty) \cap (-3,\frac{5}{3}] = [\frac{9}{11},\frac{5}{3}]$.
$$\boxed{x \in \left[\tfrac{9}{11},\,\tfrac{5}{3}\right]}$$
Solución
Ambos extremos incluidos por desigualdad no estricta.

Solución — c) $\ \left|\dfrac{x+3}{x+16}\right| > \dfrac{3}{x-4}$

Desarrollo
Justificación
$$\frac{|x+3|}{|x+16|} > \frac{3}{x-4}$$
Ejercicio 19
$\left|\frac{A}{B}\right| = \frac{|A|}{|B|}$. Dominio: $x \neq -16$, $x \neq 4$.

Caso A: $x > 4$ (lado derecho positivo).

$$|x+3|\cdot(x-4) > 3|x+16|$$

Para $x > 4$: $x+16 > 0$ y $x+3 > 0$, así:

$$(x+3)(x-4) > 3(x+16)$$ $$x^2-x-12 > 3x+48 \;\Rightarrow\; x^2-4x-60 > 0$$ $$(x-10)(x+6) > 0 \;\Rightarrow\; x < -6 \text{ o } x > 10$$

Intersección con $x>4$: $x > 10$.

Caso $x > 4$
El lado derecho es positivo. Multiplicamos por $|x+16|(x-4) > 0$, el sentido se conserva. Para $x > 4$, ambos $x+3 > 0$ y $x+16 > 0$.

Caso B: $x < 4$ (lado derecho negativo o nulo).

El lado izquierdo $\frac{|x+3|}{|x+16|} \geq 0$ siempre.

Si $\frac{3}{x-4} < 0$ (es decir $x < 4$, $x\neq-16$): la desigualdad $(\geq 0) > (\text{negativo})$ se cumple siempre, excepto cuando el lado izquierdo es $0$, es decir $x = -3$.

En $x=-3$: izq $= 0$ y der $= \frac{3}{-7} < 0$, así $0 > \frac{3}{-7}$ ✓.

Caso $x < 4$
Todo no negativo supera a cualquier negativo. La inecuación se cumple para todo $x < 4$ excepto $x = -16$ (denominador nulo).
$$\boxed{x \in (-\infty,-16)\cup(-16,4)\cup(10,+\infty)}$$
Unión de casos
Caso B da $x < 4$, $x\neq -16$. Caso A agrega $x > 10$. $x = 4$ excluido por dominio.

Solución — d) $\ \left|\dfrac{x}{1+x}\right| \geq \dfrac{1}{x}$

Desarrollo
Justificación
$$\frac{|x|}{|1+x|} \geq \frac{1}{x}$$
Ejercicio 19 + dominio
Dominio: $x \neq 0$, $x \neq -1$.

Caso A: $x > 0$.

$$\frac{x}{1+x} \geq \frac{1}{x} \;\Rightarrow\; \frac{x}{1+x} - \frac{1}{x} \geq 0$$ $$\frac{x^2-(1+x)}{x(1+x)} \geq 0 \;\Rightarrow\; \frac{x^2-x-1}{x(1+x)} \geq 0$$

Raíces de $x^2-x-1=0$: $x=\frac{1\pm\sqrt5}{2}$.

Para $x>0$: $x+1>0$ y $x>0$, así el signo depende de $x^2-x-1$.

$x^2-x-1\geq 0$ para $x \geq \frac{1+\sqrt5}{2}$.

Intersección con $x>0$: $x \geq \frac{1+\sqrt5}{2}$.

Caso $x > 0$
Para $x>0$: $|x|=x$ y $|1+x|=1+x$. Pasamos todo al mismo lado y analizamos signos. La raíz positiva de $x^2-x-1$ es $\frac{1+\sqrt5}{2} \approx 1.618$.

Caso B: $-1 < x < 0$.

$$\frac{-x}{1+x} \geq \frac{1}{x}$$

$\frac{1}{x} < 0$ y $\frac{-x}{1+x} > 0$, así la desigualdad se cumple siempre.

Caso $-1 < x < 0$
$|x|=-x>0$ y $|1+x|=1+x>0$, así izq $> 0$. Der $= \frac{1}{x} < 0$. Positivo $\geq$ negativo: siempre verdadero.

Caso C: $x < -1$.

$$\frac{-x}{-(1+x)} = \frac{x}{1+x} \geq \frac{1}{x}$$ $$\frac{x}{1+x} - \frac{1}{x} \geq 0 \;\Rightarrow\; \frac{x^2-x-1}{x(1+x)} \geq 0$$

Para $x<-1$: $x<0$ y $1+x<0$, así $x(1+x)>0$.

Signo del cociente = signo de $x^2-x-1$.

Para $x<-1$: $x^2-x-1 > 0$ siempre (raíz negativa $\frac{1-\sqrt5}{2}\approx -0.618 > -1$).

Intersección: todo $x < -1$.

Caso $x < -1$
$|x|=-x$ y $|1+x|=-(1+x)$. Para $x < -1$ la raíz negativa $\frac{1-\sqrt5}{2} \approx -0.618$ es mayor que $-1$, así que $x^2-x-1>0$ en todo el intervalo.
$$\boxed{x \in (-\infty,-1)\cup(-1,0)\cup\left[\tfrac{1+\sqrt5}{2},+\infty\right)}$$
Unión de los tres casos
Casos B y C dan $x\in(-\infty,-1)\cup(-1,0)$. Caso A agrega $x\geq\frac{1+\sqrt5}{2}$. $x=0$ y $x=-1$ excluidos por dominio.
Resultado.
a) $x \in \left(-\infty,\,\dfrac{10}{9}\right)\cup\left(\dfrac{3}{2},\,2\right)\cup\left(2,\,+\infty\right)$.

b) $x \in \left[\dfrac{9}{11},\,\dfrac{5}{3}\right]$.

c) $x \in (-\infty,-16)\cup(-16,4)\cup(10,+\infty)$.

d) $x \in (-\infty,-1)\cup(-1,0)\cup\left[\dfrac{1+\sqrt{5}}{2},\,+\infty\right)$.
