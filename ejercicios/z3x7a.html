Ejercicio

Demuestre que:

a) Si $|x+2| < 10$ y $x > 0$, entonces $|x-5| < 5$.

b) Si $|x-1| < 1$, entonces $3 < x+4 < 8$.

c) Si $|x-4| < 1$, entonces $\dfrac{1}{3} < \dfrac{1}{x-2} < 1$.

Análisis

Cada inciso es una implicación: dadas ciertas hipótesis sobre $x$, hay que concluir una cota. La estrategia es manipular algebraicamente la hipótesis hasta obtener exactamente la forma pedida en la conclusión. La desigualdad triangular $|A+B| \leq |A|+|B|$ y su variante $|A|-|B| \leq |A-B|$ son los instrumentos principales.

Variante útil de la triangular: $|x-c| = |(x-a)+(a-c)| \leq |x-a|+|a-c|$. Si controlamos $|x-a|$, podemos acotar $|x-c|$.

Solución — parte a)

Desarrollo
Justificación
$$|x+2| < 10 \;\Rightarrow\; -10 < x+2 < 10 \;\Rightarrow\; -12 < x < 8$$
Hipótesis 1 desarrollada
$|E| < 10$ equivale a $-10 < E < 10$. Restamos $2$.
$$x > 0 \quad\text{(hipótesis 2)}$$ $$\Rightarrow\; 0 < x < 8$$
Intersección de hipótesis
Combinamos $-12 < x < 8$ con $x > 0$.
$$0 < x < 8 \;\Rightarrow\; -5 < x-5 < 3$$
Restar 5
Restamos $5$ en toda la cadena para obtener $x-5$.
$$-5 < x-5 < 3 \;\Rightarrow\; |x-5| < 5$$
Conclusión
Si $-5 < x-5 < 3$, entonces en particular $|x-5| < 5$ (porque $-5 < x-5$ y $x-5 < 3 < 5$, así $|x-5| = \max(x-5, -(x-5)) < 5$).
$$\boxed{|x-5| < 5}$$
Demostrado
Ambas hipótesis fueron usadas: $x > 0$ eliminó la parte negativa, $|x+2|<10$ acotó superiormente.

Solución — parte b)

Desarrollo
Justificación
$$|x-1| < 1 \;\Rightarrow\; -1 < x-1 < 1 \;\Rightarrow\; 0 < x < 2$$
Hipótesis desarrollada
Sumamos $1$ en toda la cadena.
$$0 < x < 2 \;\Rightarrow\; 4 < x+4 < 6$$
Suma de 4
Sumamos $4$ en toda la cadena.
$$4 < x+4 < 6 \;\Rightarrow\; 3 < x+4 < 8$$
Ampliar la cota
Como $3 < 4$ y $6 < 8$, la cota $3 < x+4 < 8$ es más débil pero igualmente válida. El enunciado pide exactamente esta forma más amplia.
$$\boxed{3 < x+4 < 8}$$
Demostrado
Directamente de la hipótesis, sumando $4$ y ampliando la cota.

Solución — parte c)

Desarrollo
Justificación
$$|x-4| < 1 \;\Rightarrow\; -1 < x-4 < 1 \;\Rightarrow\; 3 < x < 5$$
Hipótesis desarrollada
Sumamos $4$ en toda la cadena.
$$3 < x < 5 \;\Rightarrow\; 1 < x-2 < 3$$
Restar 2
Restamos $2$. Notar que $x-2 > 1 > 0$, así que $x-2$ es positivo en todo el intervalo.
$$1 < x-2 < 3$$ $$\Rightarrow \frac{1}{3} < \frac{1}{x-2} < \frac{1}{1} = 1$$
Tomar recíprocos
Como $0 < 1 < x-2 < 3$, al tomar recíprocos el orden se invierte: $\frac{1}{3} < \frac{1}{x-2} < 1$.
$$\boxed{\frac{1}{3} < \frac{1}{x-2} < 1}$$
Demostrado
La cadena es directa una vez que se establece que $x-2 \in (1,3)$.
Resultado. Las tres implicaciones se demuestran manipulando algebraicamente las hipótesis hasta llegar a la forma pedida:

a) $|x+2|<10$ y $x>0$ $\Rightarrow$ $0 < x < 8$ $\Rightarrow$ $\boxed{|x-5|<5}$.

b) $|x-1|<1$ $\Rightarrow$ $0 < x < 2$ $\Rightarrow$ $4 < x+4 < 6$ $\Rightarrow$ $\boxed{3 < x+4 < 8}$.

c) $|x-4|<1$ $\Rightarrow$ $1 < x-2 < 3$ $\Rightarrow$ $\boxed{\dfrac{1}{3} < \dfrac{1}{x-2} < 1}$.
