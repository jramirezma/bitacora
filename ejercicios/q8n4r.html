Ejercicio

Resuelva:

a) $|4x+3| = 7$.    b) $|4x+5| = 2x+3$.    c) $|x^2-4| = -2x+4$.    d) $|3x-4| = -4$.

Análisis

Una ecuación $|E| = F$ tiene soluciones solo si $F \geq 0$, porque el valor absoluto nunca es negativo. Si $F \geq 0$, la ecuación se desdobla en dos casos: $E = F$ o $E = -F$. En cada caso se resuelve la ecuación lineal o cuadrática resultante y se verifica que la solución sea consistente con el caso asumido.

Siempre verificar las soluciones en la ecuación original. Una raíz del caso $E = F$ puede no satisfacer la condición $E \geq 0$ que ese caso implica.

Solución — a) $\ |4x+3| = 7$

Desarrollo
Justificación
$$4x+3 = 7 \quad \text{o} \quad 4x+3 = -7$$
Desdoblamiento
$|E| = 7$ implica $E = 7$ o $E = -7$. El lado derecho $7 > 0$, así que hay soluciones.
$$4x = 4 \;\Rightarrow\; x = 1$$ $$4x = -10 \;\Rightarrow\; x = -\frac{10}{4} = -\frac{5}{2}$$
Resolución de cada caso
Despejamos $x$ en cada ecuación lineal.
$$\boxed{x = 1 \quad \text{o} \quad x = -\tfrac{5}{2}}$$
Verificación
$|4(1)+3| = |7| = 7$ ✓. $|4(-\frac{5}{2})+3| = |-7| = 7$ ✓.

Solución — b) $\ |4x+5| = 2x+3$

Desarrollo
Justificación
$$2x + 3 \geq 0 \;\Rightarrow\; x \geq -\tfrac{3}{2}$$
Condición necesaria
El lado derecho debe ser $\geq 0$ para que la ecuación tenga solución.

Caso 1: $4x+5 \geq 0$, es decir $x \geq -\frac{5}{4}$.

$$4x+5 = 2x+3 \;\Rightarrow\; 2x = -2 \;\Rightarrow\; x = -1$$
Caso $4x+5 \geq 0$
$x = -1 \geq -\frac{5}{4}$ ✓. Verificamos: $|4(-1)+5| = |1| = 1$ y $2(-1)+3 = 1$ ✓.

Caso 2: $4x+5 < 0$, es decir $x < -\frac{5}{4}$.

$$-(4x+5) = 2x+3 \;\Rightarrow\; -4x-5 = 2x+3 \;\Rightarrow\; x = -\frac{4}{3}$$
Caso $4x+5 < 0$
$x = -\frac{4}{3} \approx -1.33 < -\frac{5}{4} = -1.25$ ✓. Verificamos: $|4(-\frac{4}{3})+5| = |-\frac{1}{3}| = \frac{1}{3}$ y $2(-\frac{4}{3})+3 = \frac{1}{3}$ ✓.
$$\boxed{x = -1 \quad \text{o} \quad x = -\tfrac{4}{3}}$$
Solución
Ambas soluciones satisfacen $x \geq -\frac{3}{2}$ y sus respectivos casos.

Solución — c) $\ |x^2-4| = -2x+4$

Desarrollo
Justificación
$$-2x+4 \geq 0 \;\Rightarrow\; x \leq 2$$
Condición necesaria
El lado derecho debe ser $\geq 0$.

Caso 1: $x^2-4 \geq 0$, es decir $x \leq -2$ o $x \geq 2$.

Combinado con $x \leq 2$: $x \leq -2$ o $x = 2$.

$$x^2-4 = -2x+4 \;\Rightarrow\; x^2+2x-8=0$$ $$(x+4)(x-2)=0 \;\Rightarrow\; x=-4 \text{ o } x=2$$
Caso $x^2 - 4 \geq 0$
$x = -4 \leq -2$ ✓. $x = 2$: límite del caso, verificamos $|4-4|=0$ y $-4+4=0$ ✓.

Caso 2: $x^2-4 < 0$, es decir $-2 < x < 2$.

$$-(x^2-4) = -2x+4 \;\Rightarrow\; -x^2+4=-2x+4$$ $$-x^2+2x=0 \;\Rightarrow\; x(-x+2)=0$$ $$x=0 \text{ o } x=2$$
Caso $x^2 - 4 < 0$
$x=0 \in (-2,2)$ ✓. $x=2$ no pertenece al intervalo abierto, se descarta en este caso (pero ya fue incluido en el caso 1).
$$\boxed{x \in \{-4,\; 0,\; 2\}}$$
Solución
Verificación: $|-4^2-4|=12=-2(-4)+4=12$ ✓. $|0-4|=4=4$ ✓. $|4-4|=0=0$ ✓.

Solución — d) $\ |3x-4| = -4$

Desarrollo
Justificación
$$|3x-4| \geq 0 > -4$$
Valor absoluto siempre no negativo
El lado izquierdo nunca puede ser negativo, pero el lado derecho es $-4 < 0$.
$$\boxed{\text{No tiene solución.} \quad S = \emptyset}$$
Conclusión
La ecuación es imposible para todo $x \in \mathbb{R}$.
Resultado.
a) $x = 1$ o $x = -\dfrac{5}{2}$.
b) $x = -1$ o $x = -\dfrac{4}{3}$.
c) $x \in \{-4,\ 0,\ 2\}$.
d) $S = \emptyset$ (sin solución).
