El problema

Suponga que $f(x) = ax + b$ y que $f(f(f(x))) = 8x + 21$ para todo $x \in \mathbb{R}$. Calcule el valor de $a + b$.

¿De qué se trata esto?

Es el mismo principio que el ejercicio 9: calcular la composición explícitamente y luego identificar coeficientes. La diferencia es que acá la función se aplica tres veces en lugar de dos. Eso genera expresiones un poco más largas, pero la idea es exactamente la misma.

Primero calculamos $f(f(x))$ como en el ejercicio anterior, y después aplicamos $f$ una vez más al resultado.

Cuando se compone una función lineal $n$ veces, el coeficiente de $x$ siempre es $a^n$. Acá $n = 3$, así que el coeficiente de $x$ en $f(f(f(x)))$ es $a^3$. Como ese coeficiente debe ser igual a $8$, tenemos $a^3 = 8$, que da $a = 2$ directamente.

Tres veces la misma función. Cada pasada multiplica por $a$ y suma algo. Al final hay que ver qué acumuló.

Manos a la obra

Paso 1: calcular f(f(x))

Desarrollo
Justificación
$f(f(x)) = a(ax+b)+b$
Se sustituye $f(x) = ax+b$ como argumento de $f$.
$f(f(x)) = a^2x + ab + b$
Se distribuye y se agrupan los términos independientes.

Paso 2: calcular f(f(f(x)))

Desarrollo
Justificación
$f(f(f(x))) = a(a^2x + ab + b) + b$
Se sustituye $f(f(x))$ como argumento de $f$.
$f(f(f(x))) = a^3x + a^2b + ab + b$
Se distribuye $a$.
$f(f(f(x))) = a^3x + b(a^2 + a + 1)$
Se factoriza $b$ en los términos independientes.

Paso 3: identificar coeficientes

Desarrollo
Justificación
$a^3x + b(a^2+a+1) = 8x + 21$
Se iguala con la condición del enunciado.
$a^3 = 8$
Coeficientes de $x$: deben coincidir.
$b(a^2 + a + 1) = 21$
Términos independientes: deben coincidir.

Paso 4: resolver

Desarrollo
Justificación
$a = \sqrt[3]{8} = 2$
La raíz cúbica real de $8$ es $2$.
$b(4 + 2 + 1) = 21$
Se sustituye $a = 2$ en la segunda ecuación.
$7b = 21$
Se suma $4 + 2 + 1 = 7$.
$b = 3$
Se despeja $b$.
$a + b = 2 + 3 = 5$
Resultado pedido.

¿El resultado aguanta?

Con $a = 2$ y $b = 3$, la función es $f(x) = 2x + 3$. Verificamos $f(f(f(x)))$ paso a paso:

$f(x) = 2x + 3$

$f(f(x)) = 2(2x+3)+3 = 4x + 6 + 3 = 4x + 9$

$f(f(f(x))) = 2(4x+9)+3 = 8x + 18 + 3 = 8x + 21$ ✓

Resultado. Los valores son $a = 2$ y $b = 3$, por lo tanto: $$\boxed{a + b = 5}$$

¿Dónde ya existe esto?

Aplicar la misma función varias veces seguidas es lo que hacen los algoritmos iterativos: cada pasada toma el resultado de la anterior y lo transforma. Lo que calculamos acá, la composición triple de una función lineal, es el modelo más simple de ese proceso. En criptografía, algoritmos como DES aplican la misma transformación básica muchas veces seguidas, y la fortaleza del cifrado viene precisamente de esa acumulación. Entender cómo se comporta una función compuesta $n$ veces, que es lo que hicimos al factorizar $b(a^2 + a + 1)$, es la misma pregunta que se hace un criptógrafo cuando analiza cuántas rondas necesita un cifrado para ser seguro.
